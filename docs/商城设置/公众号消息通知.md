# 公众号消息通知


公众号消息通知触发场景：
1、订单支付成功
2、订单取消通知
3、发货通知
4、确认收货通知
5、代理审核通知
6、新订单提醒通知
7、申报结果通知
8、拼团成功通知
9、佣金到账通知
10、被邀约的用户首次登录商城时代理收到通知
11、提交订单未支付距离订单关闭时间还剩五分钟触发消息通知
12、订单售后审核通知
13、售后订单退款到账通知

## 二、消息通知那些场景收到通知

1、订单支付成功：用户在商城中下单商品，支付成功状态，用户会收到一条支付成功的通知消息

2、订单取消通知：用户在商城中下单商品，待支付状态，取消该订单，用户会收到一条订单取消的通知消息

3、发货通知：用户在商城中下单商品并成功支付，商户把商品成功发货，用户会收到一条商品已发货的通知消息

4、确认收货通知：用户在商城中下单商品并成功支付，用户收到商品并商城中点击确认收货时，用户会收到一条确认收货的通知消息

5、代理审核通知：当用户在直接分销和独立分销中，申请为代理，商户审核通过或者不通过，用户会收到通过或者不通过相对应的通知消息

6、新订单提醒通知：在商城管理-用户管理中找到对应的用户号，点击右侧的 接收订单消息--是，此时改商城中所有用户下单支付之后，接收订单消息--是的用户可以收到通知消息

7、申报结果通知：下单该商品中，申报成功或失败会收到相应的申报结果消息通知

8、拼团成功通知：用户参与拼团活动，拼团成功之后会收到拼团成功消息通知

9、佣金到账通知：当用户存在一定佣金时，提现佣金，佣金成功到账会收到相应的消息通知

10、被邀约的用户首次登录商城时代理收到通知：

11、提交订单未支付距离订单关闭时间还剩五分钟触发消息通知：用户在商城中下单商品，待支付状态，此订单离关闭时间还剩五分钟会触发消息通知

12、订单售后审核通知：用户在商城中下单的订单进行售后，会收到该消息通知

13、售后订单退款到账通知：商城中下单的订单售后退款成功会收到该消息通知
