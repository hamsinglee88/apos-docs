## 移动商城设置

> 文档地址：商城设置/移动商城设置.md


1. 公众号授权，使用已经经过微信认证的微信公众号的管理员微信扫码授权，授权后，可以在移动商城设置页面下面配置公众号的菜单，用户在商城登录注册时，用户可以绑定微信信息，只有用户信息绑定了微信，才可以使用到微信支付
2. 设置公众号菜单，前面已经授权公众号后，可以设置公众号菜单，具体一级菜单可以最
   多设置三个，二级菜单最多设置五个

   ![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E7%A7%BB%E5%8A%A8%E5%95%86%E5%9F%8E%E8%AE%BE%E7%BD%AE.md%2F9626fa92-fdfe-4856-a3e9-a3c7381e577a_image.png?Expires=1815461236&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=OWFjd0EYR%2Bs8Fvh2lqVF9yPwuKw%3D&x-oss-process=image%2Fformat%2Cpng)
3. 公众号回复，可以分别设置关注回复，和关键词回复，关注回复，分别在用户关注微信
   公众号时和给公众号发送匹配到关键词的消息时，微信回复商家设置的信息

   ![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E7%A7%BB%E5%8A%A8%E5%95%86%E5%9F%8E%E8%AE%BE%E7%BD%AE.md%2F332d738d-0b5d-4dfa-95c2-45e525cd6446_image.png?Expires=1815461254&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=U9tS9x0rQ0J%2Bp9zhGaRHO%2FnB174%3D&x-oss-process=image%2Fformat%2Cpng)
4. 设置商城字体
   操作步骤：点击商城字体框，会出现下拉框，选择对应显示的字体

   ![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E7%A7%BB%E5%8A%A8%E5%95%86%E5%9F%8E%E8%AE%BE%E7%BD%AE.md%2F0832e183-a1e7-44ec-b0e2-49d257a1be67_image.png?Expires=1815461300&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=fg7bZ2gc2VixxVk%2BuT6Cxvuua44%3D&x-oss-process=image%2Fformat%2Cpng)
5. 设置小程序分享
   操作步骤：选择分享的场景，小程序进行分享时分享的页面会取对应的图片
   （1）分享小程序页面时，取分享页面的一部分作为分享图片
   （2）分享小程序页面时，取上传图片作为分享图片
   （3）分享商品详情页面时取商品详情作为分享图片，其它页面上传图片作为分享图片

   ![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E7%A7%BB%E5%8A%A8%E5%95%86%E5%9F%8E%E8%AE%BE%E7%BD%AE.md%2F1a3e81da-74c1-47c7-ba2a-1bd382d1a0cf_image.png?Expires=1815461356&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=yjGd0Y4gWE2Mgzsw1%2F6bLufagQ8%3D&x-oss-process=image%2Fformat%2Cpng)
