# 开通商城


已经有了APOS商户，如何开通一个商城，有哪些操作流程？

1. 首先需要开通“商城应用”。

   如何开通商城应用----将已有的APOS商户ID信息发给APOS客服，由APOS客户进行开通商城应用
2. 开通了商城应用后，接下来，需要进到商城应用，找到商城设置--通用设置，将商城绑定一个商户下的门店，填写商城名称等必要信息，保存成功后即生成了一个新商城![](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fcodyxiong%40jieztech.com%2F%E5%BC%80%E9%80%9A%E5%95%86%E5%9F%8E.docx%2Fef509641-3a83-4131-bca5-d5cf1add6e6c_image2.png?Expires=1786439362&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=%2FglNVqdTgk3f9SXvU05vE%2Bpn7Ow%3D&x-oss-process=image%2Fformat%2Cpng)
3. 已经开通了商城，查看商城的链接，分别点击“PC商城设置”，“移动商城设置”

   可以查看到商城链接地址
