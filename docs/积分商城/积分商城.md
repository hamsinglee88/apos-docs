# 积分商城


基本介绍：

积分商城是一个在APOS商城内的一个服务入口，在积分商城内专门展示积分类型商品，用户可在此处使用积分下单商品。
在前提条件----商家要创建“积分商品”类型的商品，且设置好积分商品的库存，并在商城上架商品下，积分商城内会展示所有上述的积分商品。积分商品分为“纯积分商品”和“积分+现金商品”，纯积分商品可以直接用积分进行下单购买，积分+现金商品需要积分+现金购买。

需要注意的是，为使下单积分商品成功，商家需要在门店（商城绑定的门店）管理的支付管理中，开启余额支付和积分兑换支付方式。
