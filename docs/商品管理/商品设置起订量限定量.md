# 如何设置起订量和限订量


如果商户针对某些销售的商品有最低的起售数量要求或者每单的最大购买数量要求，可以设置商品的起订量和限订量

首先，我们需要在：商品管理-商品设置内开启起订量和限订量开关然后保存。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommodity_settings.md%2F8fc908f1-de4d-457d-aa3a-759ff6eabc9e_image.png?Expires=1804411061&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=aVDCi56NeILtlIjHTaVAV9GyH3U%3D&x-oss-process=image%2Fformat%2Cpng)

保存后，商品就可以维护起订量和限订量了。如果是新创建的商品，可以在：商品管理-商品信息内找到对应商品点击修改，在商品详情页内进行修改。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommodity_settings.md%2Fc203b088-9ce0-489e-bcbc-587c28dc8fac_image.png?Expires=1804411776&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=xqlvth%2BleWaG2DYUxev7s4ggQYw%3D&x-oss-process=image%2Fformat%2Cpng)

如果是想修改历史商品的起订量和限订量，需要在门店管理-门店价格内找到对应商品，在商品列表最后的起订量和限订量字段内进行维护。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommodity_settings.md%2F2a4899fb-282f-4f40-ab48-fe23983214a8_image.png?Expires=1804414856&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=%2BThxjm%2FW94r3VNsFPJXkbq3q2hQ%3D&x-oss-process=image%2Fformat%2Cpng)

维护好起订量和限订量后，在商城购买对应商品时就会有相关限制了。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommodity_settings.md%2F01d75781-6500-4364-8b86-c1dc2af77234_image.png?Expires=1804414602&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=gsBwQiEiYVu4jW%2BLeDwgoQYeTNs%3D&x-oss-process=image%2Fformat%2Cpng)
