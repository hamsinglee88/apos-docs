# 如何新增商品基础信息里面的字段


## 1、添加自定义字段

如果商户针对现有系统提供的商品描述性字段不满意或者无法满足现有业务需求，可以在**商品管理-商品设置**内自行添加自定义字段。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommodity_custom_field.md%2Fb992df6b-9719-43a0-84ce-b1d2d8215932_image.png?Expires=1804583232&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=nejTeECPYcA3FfmXnlk8u%2B2HTX4%3D&x-oss-process=image%2Fformat%2Cpng)

添加自定义字段时，自定义字段的类型可选择：文本、下拉单选框、下拉多选框、日期。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommodity_custom_field.md%2F3d715b96-1d57-4fd3-b1c2-e1e07dfcd08c_image.png?Expires=1804583539&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=ow%2BrEbMOOYj80THVfveC8QdS7%2FY%3D&x-oss-process=image%2Fformat%2Cpng)

选择不同的字段类型会有不同的设置项，按照字段描述进行设置即可，设置完成后需要点击确定。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommodity_custom_field.md%2F06f9389d-0e21-41ae-bf59-e2c92bdda3cd_image.png?Expires=1804583693&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=xsxXo9zKnI5PcidVA4WuCo0TqhU%3D&x-oss-process=image%2Fformat%2Cpng)

自定义字段添加完成后，商品详情页内会体现新增的字段，商户可以按照实际需求维护。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommodity_custom_field.md%2Fc2eecf0c-99f0-4b3d-84b1-57ad2bcf36e6_image.png?Expires=1804586926&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=96Etu2X2HXq9r4s%2BSOz4fxdYMhY%3D&x-oss-process=image%2Fformat%2Cpng)
