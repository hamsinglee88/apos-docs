# 商品的有效期怎么管理


新增商品时可以维护商品的有效期，这个有效期的作用是：如果仓库开启商品有效期管理时，商品如果设置有效期，在商品做采购入库时维护了生产日期，会自动根据商品有效期计算结束日期。
