# 如何设置商品的价格模式

## 1、维护门店价

目前商户账号默认未开启商品门店价，所有商品均读取的商品信息内设置的基础零售价，但是如果商户下有多家门店，每家门店需要设置价格不一致，可以进入**商品管理-商品设置**，找到门店自定义价格设置，开启对应门店的自定义价格开关，保存成功后即可。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommdity_price_model.md%2Ff83cb641-e8cb-48c8-b520-01c3d91fd427_image.png?Expires=1804662123&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=kYxVxkzrsnJQeTMb9SsczDfT9NQ%3D&x-oss-process=image%2Fformat%2Cpng)

开启自定义价格后，对应门店就可以自由设置是使用基础价还是使用门店自定义价格了。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommdity_price_model.md%2F07f9c9bd-bc8f-4350-9aaf-56615b5894ff_image.png?Expires=1804662426&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=4SD3GVj3BpgXUV65kHQSp7aoGzo%3D&x-oss-process=image%2Fformat%2Cpng)

## 2、修改门店调价方式

APOS系统如果开启门店自定义价格后，门店零售价是可以任意修改的，如果商户对修改商品价格较为敏感，不允许随意修改价格的话，可以在**商品管理-商品设置**内修改“门店调价方式”为“规范”。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommdity_price_model.md%2F6a03f5e6-f43a-42f0-bedc-08ff88b57c08_image.png?Expires=1804666185&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=QnEk3yP6cS1Ah4H9tQiN1A3lK3A%3D&x-oss-process=image%2Fformat%2Cpng)

如果修改为规范，此时在**门店管理-门店价格**内是无法随意修改商品门店价格的了，需要使用调价单形式来进行调价。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommdity_price_model.md%2F216643ba-f207-4db3-9362-1ab5349000e6_image.png?Expires=1804666297&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=f3QeaaP9lhNelkBL4PN2GK70iIc%3D&x-oss-process=image%2Fformat%2Cpng)

## 3、如何使用调价单

进入门店管理-调价单页面后，可以在页面内新增基础调价单或者门店调价单

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommdity_price_model.md%2F0a701fb0-5f26-4fde-9c21-eb4e256a58c7_image.png?Expires=1804667280&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=IQZoNZo6eHqYlWOM79Ijcy3FXls%3D&x-oss-process=image%2Fformat%2Cpng)

调基础零售价和门店零售价的区别是，调门店零售价的时候需要选择调价的门店，调基础零售价则不需要

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommdity_price_model.md%2Fee19fe53-a39f-4d59-966d-8d93fa47c751_image.png?Expires=1804667533&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=ooD13sn5H5qob2sqY2cF1pSx9OQ%3D&x-oss-process=image%2Fformat%2Cpng)

调价时的生效时间指的是这一批调价商品的价格的生效时间，如果选择当前时间或者更早之前的时间，确认调价单后价格会即时生效。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommdity_price_model.md%2Ffe28998a-33fa-4fa0-92da-daee39f6980e_image.png?Expires=1804667384&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=CEdRuN4k1r5XM8nQCytRo1JgA0w%3D&x-oss-process=image%2Fformat%2Cpng)
