# 如何设置会员价


## 一、设置会员卡价

### 1、开启会员卡价

用户开通APOS系统后，系统默认的会员设置为：会员等级，如果想修改成会员卡价格，需要在客户中心-客户设置-通用设置内的“会员设置”内手动切换成：会员卡

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FMember_price.md%2Faab0ce77-4b32-4130-b132-7d3d9d2334f9_image.png?Expires=1804407035&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=cNyyWDQ67onSTFHsJzUB37J%2B1jU%3D&x-oss-process=image%2Fformat%2Cpng)

需要注意，如果之前商品设置过会员等级价，修改会员卡价进行保存后，之前设置的会员价格会被重置。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FMember_price.md%2F2fb3bdeb-da6e-480a-9357-e17a04110249_image.png?Expires=1804407540&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=LHGOOvjpe4AlBwGhkPLEQWWpm1I%3D&x-oss-process=image%2Fformat%2Cpng)

### 2、新增会员卡

开启会员卡价后，可以在客户中心内创建对应的会员卡，进入客户中心-会员卡，点击：新增会员卡，按照页面内容正常填写保存即可。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FMember_price.md%2F014c0a91-e284-4bda-80bf-2b7a19dcbd20_image.png?Expires=1804407976&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=tXYL5PzUgLofa%2FT72rIl8KxP310%3D&x-oss-process=image%2Fformat%2Cpng)

需要注意，新增完成的会员卡状态是禁用，需要手动点击启用来进行激活

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FMember_price.md%2F4a14df96-756c-442a-bf1b-8595a44808d8_image.png?Expires=1804408912&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=xnTN0sfBu8yYRd83etcLeVdN0DY%3D&x-oss-process=image%2Fformat%2Cpng)

### 3、设置商品的会员卡价

设置会员卡价前，我们需要检查下商品设置-价格设置内的会员价格是否选择的是：会员卡价，如果不是，则需要手动修改成会员卡价后保存

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FMember_price.md%2F207a965a-7fc4-4550-b88a-9f9f6e225ae5_image.png?Expires=1804409182&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=7RqAg3JuTpS0aavd6zzgabOegGM%3D&x-oss-process=image%2Fformat%2Cpng)

进入商品列表，找到需要修改的商品点击修改，在商品详情页内开启会员卡价后，即可设置对应商品的会员卡价格了。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FMember_price.md%2Fa3911f09-d3eb-4def-9f39-36d13d2b2338_image.png?Expires=1804409320&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=31U%2BafRuiV3DPTRKWGJajQA%2FVnY%3D&x-oss-process=image%2Fformat%2Cpng)

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FMember_price.md%2F4ee261ad-7710-40a8-9db0-6bf2ed1f6aed_image.png?Expires=1804409346&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=8VixpRMope%2B7wCxT%2F7OXwr0Ktb4%3D&x-oss-process=image%2Fformat%2Cpng)

如果多个商品均需要修改，只需要在商品导入内重新下载最新的导入模板，在导入模板内填写商品的会员卡价后导入即可。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FMember_price.md%2F6ab578bd-2402-4c2d-b198-70bd36f37b61_image.png?Expires=1804409427&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=AQXuSNco6EHL2NOlNhVGsPiorss%3D&x-oss-process=image%2Fformat%2Cpng)

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FMember_price.md%2F01709972-7a0e-454d-a70a-22cd68a1674a_image.png?Expires=1804409447&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=ejVUDLoK22O%2B0qtssgF1d%2BNFLAs%3D&x-oss-process=image%2Fformat%2Cpng)

## 二、设置会员等级价

### 1、开启会员等级

注册APOS商户账号后，系统默认的会员设置就已经是会员等级了，如果商户之前使用的会员卡价，就只需要参考（一、1）的内容进行切换，切换成会员等级价即可。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FMember_price.md%2F052160d1-f831-4802-bd19-d7c13bad670d_image.png?Expires=1804409737&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=adyllEZTgUUibw2NtKVQqaT%2F6fc%3D&x-oss-process=image%2Fformat%2Cpng)

### 2、新增会员等级

切换成会员等级后，可以在会员等级页面内维护所需要的会员等级，系统默认会有一个基础等级的会员等级，可以对这个会员等级的名称进行修改，但是无法删除。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FMember_price.md%2Fedeb053a-cadc-4c74-bae6-aab340acd7da_image.png?Expires=1804409880&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=XHneVafBacxU6rfYQwzZxWf00aA%3D&x-oss-process=image%2Fformat%2Cpng)

## 三、不想设置会员价的操作方法

如果您不想设置任何会员价，可以进入**商品管理-商品设置**，修改价格设置-会员价格的设置项为“不设置”。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FMember_price.md%2F1b829cce-f87a-4621-8a01-8223ac009a5b_image.png?Expires=1804661934&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=gD3toKO%2F013tAHtbHrGBPLM6Uug%3D&x-oss-process=image%2Fformat%2Cpng)
