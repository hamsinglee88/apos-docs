# 如何维护商品成本价


可以在商品管理-商品信息内点击修改商品，在商品详情页内勾选“最新进价”后，维护最新进价的金额。最新进价的金额就是当前商品的成本价。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommdity_costprice.md%2Fb97d356f-45f4-4601-aefc-36eb52f81d8b_image.png?Expires=1804925601&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=GKx2tLRGwllewh2wGRJvILlRu1U%3D&x-oss-process=image%2Fformat%2Cpng)

维护商品最新进价后，如果商品做采购入库，商品的成本价会被自动带出。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommdity_costprice.md%2F5324ec30-2c1c-4c91-91f0-a2279c770324_image.png?Expires=1804925560&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=7nBLjmWL9%2FeQpXeLTtJCQKsjsuw%3D&x-oss-process=image%2Fformat%2Cpng)

确认入库后，商品的成本价就可以在商品库存内体现出来了。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommdity_costprice.md%2Fba0e30f0-ac6b-482d-ab20-c4065864d59b_image.png?Expires=1804928227&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=6XTBcyFqum2y8m3F9aUhjbOyo1Y%3D&x-oss-process=image%2Fformat%2Cpng)
