# 批量导入商品


需要批量导入商品时首先要进入“商品导入”菜单，选择“导入商品综合资料”，下载模板；

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fcodyxiong%40jieztech.com%2Fcommodity-management.md%2F61bc1ea6-810e-41e7-9cfb-b064e789f6d1_image.png?Expires=1794823189&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=XG4wWApaZytOxDPaTAsdhe38k3g%3D&x-oss-process=image%2Fformat%2Cpng)

根据模板表格内的指引填写数据，红色*号字段为必填；

表格支持填写单规格和多规格的商品信息，多规格商品数据最多只能填写三种规格类型的信息，但规格值的数量不做限制；

其它表格填写注意事项请查看导入模板文件中的使用说明
