# 如何维护商品税率

## 1、维护单个商品税率

如果需要维护单个商品的商品税率，可以进入商品详情页，在税率信息内针对商品对应的税则维护税率。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommodity_tax.md%2Faf71f2b2-6a4b-4bad-94ff-651a0da68741_image.png?Expires=1804562625&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=I7AnGB4HD%2BGjiSXkyADxLriYqaI%3D&x-oss-process=image%2Fformat%2Cpng)

## 2、维护商品分类的税率

如果同一商品分类下的商品税率均一致，可以在**商品管理-分类信息**内，针对分类设置税率

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommodity_tax.md%2F7eb5e52d-224e-4e56-99b6-faa3c6bff495_image.png?Expires=1804563015&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=aL8kCRBRgsa0%2F1F0Krsqycw25HQ%3D&x-oss-process=image%2Fformat%2Cpng)

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommodity_tax.md%2Fcc0e941b-f246-4146-8d01-89844a3f55bc_image.png?Expires=1804563036&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=2o0DBg1Y0aepk10o%2F5i44aqGFPU%3D&x-oss-process=image%2Fformat%2Cpng)

如果分类设置税率后，后续新增/修改商品的分类为该分类时，会自动带出当前分类设置的税率。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommodity_tax.md%2F456a3a53-bd75-4437-8b11-499b308e41e2_image.png?Expires=1804563126&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=Ee3kQr%2BiMV9OlGp4cAecwIjetzo%3D&x-oss-process=image%2Fformat%2Cpng)

**注:如果此时修改商品分类为未设置税率的分类，商品的税率会被清空。**

## 3、批量修改商品税率

如果商户下的多个商品都想要修改商品税率，可以先进入**商品管理-商品信息**，点击页面内的导出，导出后我们可以在最后几列查看到对应商品现在的税率。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommodity_tax.md%2F265d3cbc-fa2c-44c7-b296-9493f2584d83_image.png?Expires=1804563699&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=oRsR15VtGhpLQq4qYCqCRaHNBHc%3D&x-oss-process=image%2Fformat%2Cpng)

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2Fcommodity_tax.md%2Fae6054dc-14bc-4777-8266-d5bf39ec750a_image.png?Expires=1804563879&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=jo4PmKEUncOiVWUypZMpsz6Zkco%3D&x-oss-process=image%2Fformat%2Cpng)

假设需要修改商品的跨境综合税为9.1%，只需要在模板内修改跨境综合税的字段填写**9.1**；

又假如需要修改商品的跨境综合税为0，只需要在模板内修改跨境综合税的字段填写**0**。

**接下来找到“批量修改商品信息”菜单，点击该菜单后，在弹框中点击“下载模板”，再将之前的商品信息复制到专门导入用的excel中，确认商品信息无误后，再点击导入弹框界面，点击“选择文件”，选中前面的导入模板excel进行导入即可。注意：导入的文件一定要用导入界面下单的模板excel 导入。**


![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Fcommodity_tax.md%2Ffb084ec6-7827-4807-9fdb-32fa95101c0a_image.png?Expires=1823928510&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=c09NN9kq3hVKNKeKgQV05NNxsSo%3D&x-oss-process=image%2Fformat%2Cpng)

需要注意，不同税则可填写或者可导入的税率是固定的，如果需要填写的税率不在可选的范围内，可以联系**APOS客服**进行添加维护。
