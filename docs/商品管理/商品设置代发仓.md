# 代发业务如何设置商品发货仓库

## 1、启用代发应用

商户如果要走代发业务，需要开启代发多仓的应用，可以在应用商店内找到并且开通

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FCommodity_delivery_warehouse.md%2F65af6fac-25bc-4abf-a9de-c63f1cc0a270_image.png?Expires=1804416010&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=a8ESYfZSD%2FexsrlB7AJpMlEU7Ek%3D&x-oss-process=image%2Fformat%2Cpng)

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FCommodity_delivery_warehouse.md%2F7f23782c-b094-447a-b4df-0dbab1bdc6e3_image.png?Expires=1804415990&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=V0UbjifepSHwVwKc3ePhErkI%2BlQ%3D&x-oss-process=image%2Fformat%2Cpng)

开启后如果应用内没有展示相关设置项，请联系**APOS客服**或者**APOS实施**进行基础设置项的设置，设置完成后，即可进行代发仓的设置了。

## 2、设置代发仓

商户可以根据实际要走的代发仓进行代发仓的设置，设置项均需要按照实际业务填写。其中“货主”字段可以理解成这个代发仓的名称。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FCommodity_delivery_warehouse.md%2F760749d1-56c9-45e4-900f-6e08d4f8ca17_image.png?Expires=1804416167&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=f2UaSKe0IFEmo49deHi0W5kXxaY%3D&x-oss-process=image%2Fformat%2Cpng)

## 3、商品设置发货仓库

维护好代发仓信息后，就可以在商品信息内维护商品的发货信息了，进入：商品管理-商品信息，找到要设置的商品，进入商品详情页，在详情页内的SKU维度会展示仓库字段，可以选择当前商户使用的代发仓，如果商户有多个代发仓，则可以下拉选择对应商品的代发仓。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FCommodity_delivery_warehouse.md%2F5c1ac0ce-1e2a-45f3-af53-69dbf07bf9a5_image.png?Expires=1804416472&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=Vv1Ra7AevDq%2Bt4NCngUi1fNWD64%3D&x-oss-process=image%2Fformat%2Cpng)

设置好代发仓后，需要设置商品在对应代发仓的第三方系统ID，第三方系统ID可以理解成这个商品在对应代发仓内的商品编码/货号，如果设置了仓库，则第三方系统ID是必填项。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FCommodity_delivery_warehouse.md%2F2492010b-683b-4eaa-b842-2e47fe217335_image.png?Expires=1804416570&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=E47z1XN4vURQo12XA%2B8m%2FVGoaao%3D&x-oss-process=image%2Fformat%2Cpng)

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FCommodity_delivery_warehouse.md%2Fe0e460d4-4f8a-4aa4-baa4-7f694c9a5e9b_image.png?Expires=1804416600&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=08%2F%2B4jJdQeq5i6BU5UIOQgB6YiE%3D&x-oss-process=image%2Fformat%2Cpng)

如果是大量商品需要设置发货仓库，可以在：商品管理-商品导入内，找到“批量修改外部仓”功能，下载导入模板后按照模板要求填写导入即可。
