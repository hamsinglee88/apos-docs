# 分销团长

分销团长
========

说明：支持独立分销商设置其店铺的会员为团长身份，团长分享独立分销商的店铺发展新用户，新用户的订单数据记录在该团长下，独立分销商再自行与团长之间线下结算佣金

指引：

1、开启独立分销功能，入口：商城设置-分销设置

2、开启“团长应用”，入口：应用商店-商店首页-分销团长

3、独立分销商进入商户商城，会显示“会员管理”入口（下图一）

![](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fcodyxiong%40jieztech.com%2F%E5%88%86%E9%94%80%E5%9B%A2%E9%95%BF.docx%2Fc980bfe1-f5f2-410c-8621-6b9e8e7c8ae4_image1.png?Expires=1786499817&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=0%2BtDW5diuhXIEFVf6bO%2BiWFA%2Buo%3D&x-oss-process=image%2Fformat%2Cpng)

3、会员管理列表展示在子店铺注册的所有会员（包括独立分销商直接发展和团长发展的）

4、“设为团长”后会员带有团长标签，显示订单数量（统计的是该团长发展的所有会员产生的所有状态的订单数量，不包含团长自己）

![](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fcodyxiong%40jieztech.com%2F%E5%88%86%E9%94%80%E5%9B%A2%E9%95%BF.docx%2F8abf0158-a5fa-4a6c-ad44-b373165e80ee_image2.png?Expires=1786499817&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=WyhB8ePLb%2BKjrCYMibs0UuCTPds%3D&x-oss-process=image%2Fformat%2Cpng)

5、点击订单数量进入订单详情列表（上图三），“累计订单数”符合筛选条件的订单总数，“累计订单金额”符合筛选条件的订单总实付金额

6、“取消团长”后团长标签和订单数量隐藏，不可查看订单数据

7、如果“团长应用”关闭，则团长标签，设为/取消团长和订单数量按钮都隐藏

![](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fcodyxiong%40jieztech.com%2F%E5%88%86%E9%94%80%E5%9B%A2%E9%95%BF.docx%2Fb4755692-85b4-4303-b2bb-6e553900598b_image3.png?Expires=1786499817&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=PDr%2BXNJtx%2BcBTFjXuMPhFrp%2FaVs%3D&x-oss-process=image%2Fformat%2Cpng)

8、开启团长应用，后台列表显示所有团长信息，支持导出

【订单总数量】统计该团长发展的会员产生的订单总数量（所有状态的订单）

【订单总金额】统计该团长发展的会员产生的订单总实付金额（所有状态订单）

![](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fcodyxiong%40jieztech.com%2F%E5%88%86%E9%94%80%E5%9B%A2%E9%95%BF.docx%2Fae8b1ec0-0952-41fc-baf8-ae0d403c7dff_image4.png?Expires=1786499817&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=IC%2BgdV1PHvuYmfPbqI4jMjLYXtM%3D&x-oss-process=image%2Fformat%2Cpng)

9、点击“查看订单详情”可查看该团长下所有会员产生的订单信息（所有状态订单），支持导出操作

【订单金额】该笔订单的实付金额

![](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fcodyxiong%40jieztech.com%2F%E5%88%86%E9%94%80%E5%9B%A2%E9%95%BF.docx%2Fa50502b1-0097-4bfb-be4f-6c30659fc8db_image5.png?Expires=1786499817&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=7cql%2Bt3wIg4Lnngz7yp2xGGZkIc%3D&x-oss-process=image%2Fformat%2Cpng)

说明：

独立分销商可设置任意会员为团长，比如团长A下面有B会员，那B升级为团长之后，B上级还是团长A，且B所创建的订单数据还是会记录在团长A下，如B发展了会员C，
那会员C的订单数据会记录给团长B，不会记录给团长A（订单数据只记录一层关系）
