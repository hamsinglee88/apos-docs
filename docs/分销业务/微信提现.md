# **微信提现操作手册**

## 一、微信提现基础设置

使用微信提现需要有个大前提：在微信支付商户后台内开通：企业付款到零钱功能，

![图片1.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FWechat%20withdrawal.md%2F78b71ce5-0bba-46f6-8fc7-b8f1316e8528_%E5%9B%BE%E7%89%871.png?Expires=1797329564&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=%2BPsTr5edzdnncGj3Jk%2BrZ2vByek%3D&x-oss-process=image%2Fformat%2Cpng)

在微信支付商户平台添加“企业付款到账户”能力后，可以支持在微信公众号、小程序提现。

1、APOS商户后台配置

进入APOS后台后，点击右上角应用商店-已开通应用，进入商城应用后，进入提现设置tab页，页面内开启微信提现按钮，并且填写如下内容后点击保存。

![图片2.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FWechat%20withdrawal.md%2F932c1c13-9aba-4c9b-835e-47fab293a555_%E5%9B%BE%E7%89%872.png?Expires=1797329578&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=cQCutwlwQdw7CrgFds2JIFYHeXI%3D&x-oss-process=image%2Fformat%2Cpng)

成功配置微信提现需要设置微信支付的商户号、秘钥、导入证书、填写证书密码。其中商户号可以在微信支付商户后台-账户中心-商户信息-基本账户信息内找到。

![图片3.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FWechat%20withdrawal.md%2F7b2340d7-e486-4487-8dda-a339d2667c76_%E5%9B%BE%E7%89%873.png?Expires=1797329590&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=LkKcpZmldQCCzkyONFMYbI3gjT8%3D&x-oss-process=image%2Fformat%2Cpng)

微信支付秘钥和证书可以在微信支付商户后台-账户中心-API安全内获取

![图片4.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FWechat%20withdrawal.md%2Fd040684c-3179-457d-9d8b-416106a72838_%E5%9B%BE%E7%89%874.png?Expires=1797329600&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=GACpye%2FwL%2BoauMcuZPRB9OyT57w%3D&x-oss-process=image%2Fformat%2Cpng)

证书密码则是生成证书时设置的密码，如果不记得证书密码，可以通过双击证书文件安装，安装时输入密码方式测试正确性，双击证书文件后，进入导入向导，点击下一步。

2、自动审核和自动发放设置

开启微信提现后，会展示自动审核和自动发放功能，这2个功能默认选择的是“人工审核”，如果为人工审核，用户发起微信提现后，需要人工的对用户的提现申请进行审核和发放。

修改为自动审核和自动发放的话，如果用户的微信提现金额**小于等于**自动审核设置的金额，则会自动进行审核通过和发放，也可单独设置自动审核或者自动发放。

![图片10.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FWechat%20withdrawal.md%2F70eb0eb2-8626-4145-b32a-f32b6a71d019_%E5%9B%BE%E7%89%8710.png?Expires=1797329662&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=wRHukKLUSjccmOxTEpxslZESvTw%3D&x-oss-process=image%2Fformat%2Cpng)

自动审核、发放和手动审核发放的区别是，如果是自动审核和自动发放，系统会有备注“自动审核通过”、“佣金提现自动发放”。

![图片11.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FWechat%20withdrawal.md%2Fdc8af020-f4dd-4a8e-9a9b-25efab37a2cf_%E5%9B%BE%E7%89%8711.png?Expires=1797329683&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=Uee5URys6hBpijuHrg%2FYOEVQF0M%3D&x-oss-process=image%2Fformat%2Cpng)

## 二、如何进行微信提现

1、首先进入商城-我的，点击下图内2个红框的入口，均可进入提现页面

![图片12.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FWechat%20withdrawal.md%2F21b70088-d16a-4df9-b52c-42f2cb59765b_%E5%9B%BE%E7%89%8712.png?Expires=1797329693&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=EuneNj3EuWs1EUXWvj0WLAuDjnw%3D&x-oss-process=image%2Fformat%2Cpng)

![图片13.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FWechat%20withdrawal.md%2F3d29474e-3ffa-4811-b291-841b1ff146e7_%E5%9B%BE%E7%89%8713.png?Expires=1797329699&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=U%2F4MYGFoZGKn4hqwHqOm%2FxpzbtM%3D&x-oss-process=image%2Fformat%2Cpng)

2、在立即提现页面内选择提现方式为“微信提现”，然后填写金额即可发起提现（提现金额范围和每日提现次数可以在商城后台-商城应用-提现设置内进行调整）

![图片14.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FWechat%20withdrawal.md%2F97f3c956-4fba-4542-8885-6a2028bd2d72_%E5%9B%BE%E7%89%8714.png?Expires=1797329713&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=taaiA4TwpnRxGx8trbgorkuqrSQ%3D&x-oss-process=image%2Fformat%2Cpng)

3、如果用户反馈看不到微信提现的提现方式，则需要注意，**微信提现只支持在小程序、公众号**内操作，如果用户从**浏览器H5**进入提现页面，页面内是**不展示微信提现入口**的。此外，用户如果是使用邮箱登录小程序，并且没有在小程序内授权过，也是看不到微信提现入口的。

## 三、部分提现失败的失败原因解释

1、如果提现失败的失败原因为：商户号有误，无法查询商户信息。需要检查商城应用-提现设置内的微信提现商户号是否和微信支付内的商户号一致。

![图片15.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FWechat%20withdrawal.md%2F95e28d96-8a6a-476f-a09c-0ad9c502618d_%E5%9B%BE%E7%89%8715.png?Expires=1797329725&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=0r%2FUl4S%2FqvLM7xRG57ilYjccnZQ%3D&x-oss-process=image%2Fformat%2Cpng)

2、如果提现失败的失败原因为：签名错误。需要检查商城应用-提现设置内的微信提现的秘钥是否正确。

![图片16.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FWechat%20withdrawal.md%2F445caa95-7ad4-4632-a3f2-d16d96ba75de_%E5%9B%BE%E7%89%8716.png?Expires=1797329733&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=I1W%2F0%2B5zCAGiY8LYll%2F6NNc3TFc%3D&x-oss-process=image%2Fformat%2Cpng)

3、如果提现失败的失败原因为：指定的网络密码不正确。需要检查商城应用-提现设置内的微信提现的证书密码是否正确，或者是上传的证书和证书密码是否匹配。

![图片17.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FWechat%20withdrawal.md%2Fa4c4f101-30bf-4ec2-89e7-8d8f0342f366_%E5%9B%BE%E7%89%8717.png?Expires=1797329744&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=hUkxBaV2cA9QrjcYZpmPqKMo0w0%3D&x-oss-process=image%2Fformat%2Cpng)

4、如果提现失败的失败原因为：找不到申请的对象。需要检查商城应用-提现设置内的微信提现的证书是否为微信支付内生成的证书。

![图片18.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FWechat%20withdrawal.md%2F5d485da6-0f39-4a59-9326-9f01efd2b6d6_%E5%9B%BE%E7%89%8718.png?Expires=1797329756&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=0g55thGE4b%2BkJcvacgLL18sWsSw%3D&x-oss-process=image%2Fformat%2Cpng)

5、如果提现失败的失败原因为：商户号与商户appid不匹配。需要检查当前提现用户的appid是否有和微信支付进行关联。

![图片19.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FWechat%20withdrawal.md%2F6819e0e0-d3f3-4373-9110-a7354ea1dc61_%E5%9B%BE%E7%89%8719.png?Expires=1797329764&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=5jWF%2FFBQEmkOCHG19EUxh6NkxmY%3D&x-oss-process=image%2Fformat%2Cpng)

6、如果当前微信支付账户内余额不足时，用户发起提现并且确认发放后，提现申请会进入”提现中”页面内，需要等到微信支付内有足够的金额后点击重新发放即可。

![图片20.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fluoyuanchuan%40jieztech.com%2FWechat%20withdrawal.md%2F9273720f-39b7-43a9-9955-1920580ce64d_%E5%9B%BE%E7%89%8720.png?Expires=1797329771&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=AnY6LfaQPI6dtybdXzowR%2BJPZhE%3D&x-oss-process=image%2Fformat%2Cpng)

7、如果微信提现时接口返回下图的错误信息，则需要在微信支付-“企业付款到零钱”内开启API付款，填写IP后等待10分钟后重新尝试提现。
