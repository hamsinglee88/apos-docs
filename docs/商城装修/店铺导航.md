# 店铺导航

基本介绍：店铺导航默认存在”首页”、”分类”、”购物车”、”我的”四个导航位
，位于商城首页底部，店铺导航可以设置显示图标选中和未选中展示效果，修改图标名称且添加图标链接，可以更好的加强用户使用商城的便利。另外商家可以自行再调价一个导航，达到五个导航。

## 1、设置装修店铺导航

步骤1：进入自定义页，点击店铺导航
