# 悬浮窗

浮动窗口可在商城首页中任意滑动以及添加选择跳转页面进行跳转功能，商户后台中悬浮窗中添加多大尺寸的图片商城中即会展示多大尺寸的图片。

## 创建悬浮窗

进入自定义页，点击页面设置

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Frenovation-new.md%2Fd0f3a5d6-2e7d-44bd-a20f-4cb359785372_image.png?Expires=1805702311&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=%2Fd6iXmaV9FBy01fXg2tEUeJXzrI%3D&x-oss-process=image%2Fformat%2Cpng)

点击浮动窗口开关按钮“开启”

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Frenovation-new.md%2F876622d5-991e-4e11-9c98-7724b444e17c_image.png?Expires=1805703238&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=SlnrrzSAHCAyfCzzcjxEkIKanhM%3D&x-oss-process=image%2Fformat%2Cpng)

点击添加悬浮窗按钮（最多可以添加三个悬浮窗）

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Frenovation-new.md%2F6a743f4a-bd18-4770-aaf0-a1919f41e640_image.png?Expires=1805703292&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=56Y4zP%2B0ExEgLlu1sA%2Bg4ER25TI%3D&x-oss-process=image%2Fformat%2Cpng)

点击+添加图片，选择电脑中已存的图片即可

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Frenovation-new.md%2Fcef7a425-395d-442b-9e2d-5615e409166b_image.png?Expires=1805703322&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=w9zPq4ZUFYBj386HNC9cczuIr7Q%3D&x-oss-process=image%2Fformat%2Cpng)

选择跳转到的页面：下拉框可选择显示”商品分类”、”营销活动”、”页面”以及自定义链接

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Frenovation-new.md%2F0e20bf55-021e-4b98-9adc-e22531d5b71a_image.png?Expires=1805703399&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=kyDu40pmbllMFGwmeZJCMVRFEig%3D&x-oss-process=image%2Fformat%2Cpng)
