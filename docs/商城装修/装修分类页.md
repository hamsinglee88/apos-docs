# 分类页

基本介绍：系统模板分类页固定会有一个分类组件，分类组件中页面样可以设置样式1 或 样式2。也可以添加其他装修组件内容在分类页中。在分类组件里，样式1可以通过多次添加分类，在添加分类选择分类的父级进行多级分类的关联，最多可以支持三级分类之间关联。样式2可以直接添加一级分类与商品关联。

需要注意的特点：

如果在装修应用--模板页面--分类页，未编辑过任何内容。那么移动商城还是使用旧的分类页（商城管理--商城装修--分类页模板内容）；

如果如果在装修应用--模板页面--分类页，编辑了任何内容保存了，那么在最新版的小程序下，将会使用在此处自定义装修的分类页内容，不会再使用旧的分类页。

另外，在系统模板的分类页，也可以自行添加其他的装修组件到页面来。

## **页面设置-样式1如下**

### 1.在分类页添加一二级分类关联：

步骤1：首先添加一个分类，作为一级分类，此时，分类名称为必填项，分类图片/分类父级/选择分类三项都保持空，点击“添加分类”，可以完成点击一个一级分类。此时还不能点击“发布”，需要至少在一级分类下有二级分类内容才能发布。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Frenovation-new.md%2Ff22f4937-d016-4535-9471-405de5316eda_image.png?Expires=1805699148&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=lxSQilxDxR1YzJUB5MPMFX%2BfKag%3D&x-oss-process=image%2Fformat%2Cpng)

步骤2：建一个分类挂在一级分类下面作为二级分类

填写二级分类的名称，分类图选填，在“分类父级”下拉框选择前面的“一级分类”，并在“选择分类”处下拉选择商城商品的具体分类内容。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Frenovation-new.md%2F6759eec9-4637-494f-899f-286d9f7aa24a_image.png?Expires=1805699568&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=RTmBwXyMWdzhtR6teesyreWDd%2Fk%3D&x-oss-process=image%2Fformat%2Cpng)

这样就完成一二级模式分类装修。

### 2.在分类页添加一二三级分类关联

如果在分类页添加一二三级分类关联，可如下操作，在上述有一级二级分类的基础上，再次新建分类，选择“分类父级”时，选择前面建好的“二级分类”，挂在其下面即作为三级分类，分类的内容在“选择分类”选项的下拉框选择具体内容即可。
