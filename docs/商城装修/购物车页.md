# 购物车页


基本介绍：购物车页可以进行添加所有组件装修商城购物车，当商城购物车没有商品时显示一张图片文案，当商城购物车中添加了商品，购物车中会显示已添加的商品，

点击商城管理--商城装修(新)--模板页面--购物车页面--编辑按钮

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E8%B4%AD%E7%89%A9%E8%BD%A6%E9%A1%B5.md%2F5139a295-504c-43f8-9719-c864bb054c72_image.png?Expires=1821517875&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=TCrjx3fi14lDMDFUt81Zr6jixmM%3D&x-oss-process=image%2Fformat%2Cpng)

编辑装修购物车页面，发布，商城购物车页面即可展示最新的装修效果
