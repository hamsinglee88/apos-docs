# 装修支付成功页


基本介绍：支付成功页为商城中支付成功的显示页面，支付成功页默认 由支付状态，支付信息，支付页跳转三个组件组成，支付成功页中可以任意添加个组件，调整组件颜色进行装修

## 1、装修支付成功页

进入商城装修应用，打开模板页面，点击“支付成功页”的“编辑”按钮

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E8%A3%85%E4%BF%AE%E6%94%AF%E4%BB%98%E6%88%90%E5%8A%9F%E9%A1%B5.md%2F053cef99-d95a-49dc-8f29-28ca2710be29_image.png?Expires=1816587182&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=eNHjvL1BHRIHaSVU8JSaCvVYKXk%3D&x-oss-process=image%2Fformat%2Cpng)

进入编辑页面之后，默认显示支付状态、支付信息、支付页跳转组件存在，需要添加其他组件以拖拽方式添加组件进行装修，（注：支付模板是无法编辑/删除的）

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E8%A3%85%E4%BF%AE%E6%94%AF%E4%BB%98%E6%88%90%E5%8A%9F%E9%A1%B5.md%2F1f3cc38f-e3db-4d6f-8428-cf2042e9468d_image.png?Expires=1819792272&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=xZ%2FuIITXw05OgOSWmzWTaXpOUMc%3D&x-oss-process=image%2Fformat%2Cpng)

支付状态、支付信息、支付页跳转组件可以设置不同颜色进行装修，点击背景色可以切换选择不同颜色，默认颜色为白色（透明色值为 rgba(255,255,255,0) ）

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E8%A3%85%E4%BF%AE%E6%94%AF%E4%BB%98%E6%88%90%E5%8A%9F%E9%A1%B5.md%2F6bc83aef-7e6f-45dd-813b-5a83c2309e44_image.png?Expires=1819792537&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=Bp7JaaVcPgnV6VImSFUC9ma%2F8Dk%3D&x-oss-process=image%2Fformat%2Cpng)
