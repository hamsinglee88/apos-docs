# 页面弹窗

> 文档地址：商城装修/页面弹框.md


基本介绍：已经登陆的情况下，每天第一次进入商城或每一次浏览商城时，会出现弹窗，弹窗中可以设置跳转链接到“商品分类”、“营销活动”、“页面”以及自定义“链接。弹窗最多可以添加三个弹窗。

## 创建页面弹窗

步骤1：点击开启弹窗开关
![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Frenovation-new.md%2Fce0c5901-7ce6-4325-9162-e32669e8bbe4_image.png?Expires=1805869030&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=NfTU5%2BiAedGdFGC0aKE4TcJCRGU%3D&x-oss-process=image%2Fformat%2Cpng)
步骤2：点击图片框，添加图片
![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Frenovation-new.md%2F4e573357-aa89-47a1-a345-8904341c09ac_image.png?Expires=1805869061&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=awSCqeBqZIV%2BgMCm16TszXjYvDw%3D&x-oss-process=image%2Fformat%2Cpng)

步骤3：点击选择跳转到页面会出现下拉框，选择需要跳转的链接
![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Frenovation-new.md%2F95cedb6d-33bc-406b-a7c3-bb6ca760f5a1_image.png?Expires=1805869096&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=qLtJhWX%2FOjjqxjp2pqJucUC%2BC%2BA%3D&x-oss-process=image%2Fformat%2Cpng)

步骤4：点击弹窗场景-每天第一次浏览商城时弹出，每天第一次进入商城即会弹出弹窗
![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Frenovation-new.md%2F587b60ba-200c-4047-9a9a-e79a3c4ab8a9_image.png?Expires=1805869123&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=rG%2BF3%2BRfpnMYT2uycT40DT%2FXUBE%3D&x-oss-process=image%2Fformat%2Cpng)

步骤5：点击弹窗场景-每次浏览商城时弹出，每次进入商城即会出弹窗

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Frenovation-new.md%2F13dac292-8452-47ac-9981-4fcc718f88ac_image.png?Expires=1805869233&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=PIDi22IzTt9yv%2FocZBUnLrnZavc%3D&x-oss-process=image%2Fformat%2Cpng)
步骤6：选择指定用户，默认选择所有用户
![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Frenovation-new.md%2F12f307e9-8718-4c4d-98d7-51879b18782d_image.png?Expires=1805869186&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=vvm2IOg%2Fd4W21UxlpftX%2F17RSJU%3D&x-oss-process=image%2Fformat%2Cpng)
步骤7：点击添加弹窗按钮，点击一次即添加一个弹窗，最多可以添加三个弹窗。
![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Frenovation-new.md%2F8b12db0a-2bc4-47a2-a90a-986d35833acf_image.png?Expires=1805869262&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=1R6ZfsQ2GlsYJoviQF2l0xXz5XA%3D&x-oss-process=image%2Fformat%2Cpng)

## 商城中查看弹窗的效果

商城中查看弹窗的效果，点击关闭第一个弹窗，会展示第二个弹窗，点击关闭第二个弹窗，会展示第三个弹窗，点击关闭第三个弹窗，显示商城首页
