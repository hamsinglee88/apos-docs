# 商城销售保税品税费方案


前提条件:

1、商品设置成保税品

2、商品设置税率

3、门店管理--门店设置--业务设置--保税品定价含跨境综合税--设置“是 / 否”

一、设置用户下单商品 含税

进入门店管理---门店设置----业务设置---保税品定价含跨境综合税 设置“是”

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E5%A6%82%E4%BD%95%E8%AE%BE%E7%BD%AE%E7%A8%8E%E8%B4%B9%E5%88%86%E7%A6%BB.md%2F34392d42-3253-4d27-88a0-6285afeec9c1_image.png?Expires=1824457074&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=TnC%2B33QK1tnnFXqsAcw2GJKB4J4%3D&x-oss-process=image%2Fformat%2Cpng)

用户下单时，确认订单页面中跨境税费不会计算显示税费，税费已经包含在商品售价中

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E5%A6%82%E4%BD%95%E8%AE%BE%E7%BD%AE%E7%A8%8E%E8%B4%B9%E5%88%86%E7%A6%BB.md%2F2bb1b26d-d65b-4e80-abcb-7b560c4b7654_image.png?Expires=1824457236&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=Ncq3GJ9rRIfBl%2Fy7N93WTF1ptWE%3D&x-oss-process=image%2Fformat%2Cpng)

二、设置用户下单商品 不含税

进入门店管理---门店设置----业务设置---保税品定价含跨境综合税 设置“否”

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E5%A6%82%E4%BD%95%E8%AE%BE%E7%BD%AE%E7%A8%8E%E8%B4%B9%E5%88%86%E7%A6%BB.md%2Fafc01a7a-32d6-465d-9e8c-0cbfc149d69a_image.png?Expires=1824457270&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=wBDSgf0jNvXjqrjGLn0vv3%2BCzW8%3D&x-oss-process=image%2Fformat%2Cpng)

用户下单时，确认订单页面中跨境税费会计算显示税费，税费没有包含在商品售价中

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E5%A6%82%E4%BD%95%E8%AE%BE%E7%BD%AE%E7%A8%8E%E8%B4%B9%E5%88%86%E7%A6%BB.md%2Fc0513245-0ee5-48f5-be9a-7d7720292939_image.png?Expires=1824457295&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=zzdK4dzF8AKz5ER09viHzjXJejY%3D&x-oss-process=image%2Fformat%2Cpng)
