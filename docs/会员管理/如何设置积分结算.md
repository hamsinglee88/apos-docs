# 设置积分结算

点击进入客户中心--会员设置--通用设置--积分设置

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E5%A6%82%E4%BD%95%E8%AE%BE%E7%BD%AE%E7%A7%AF%E5%88%86%E7%BB%93%E7%AE%97.md%2F03f10ea6-c98a-49a8-aa77-ec1ff799ff6e_image.png?Expires=1823052003&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=C4TIqQU%2BaRhtyX6XOPFpUUF7jQE%3D&x-oss-process=image%2Fformat%2Cpng)

在积分设置中设置成--积分结算在支付成功后，用户支付成功即可获取积分

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E5%A6%82%E4%BD%95%E8%AE%BE%E7%BD%AE%E7%A7%AF%E5%88%86%E7%BB%93%E7%AE%97.md%2Fa1b3e19b-9a0e-40ad-b331-8bb24215b629_image.png?Expires=1823052144&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=aZM%2BIozpV7DZLhW0nwuoEC2vptI%3D&x-oss-process=image%2Fformat%2Cpng)

在积分设置中设置成--积分结算在订单确认收货 [  ]天后。
设置 自定义 天数，用户在商城端点击 “确认收货”按钮之后，按照自定义输入的天数即可获取积分
