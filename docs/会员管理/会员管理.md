# **商城会员服务**

## **基本介绍：**

商城会员服务主要包括商家后台客户管理和商城前台会员中心两部分。其中

客户管理是提供对商城会员用户进行会员信息、会员权益进行管理的服务；会员中心是商城用户在商城可以去查看会员卡信息、领卡获取权益的服务。
对于会员管理，目前支持两种模式，会员等级模式以及会员卡模式。

如使用会员等级模式，新用户注册后默认会有一个会员等级（默认等级），商家可以在会员等级菜单去新增编辑不同的会员等级。相应的可以将商城会员用户设置为不同的会员等级的会员，同时给商品设置会员等级价格，会员就可以享受到相应尊享价。

如使用会员卡模式，商家需要先创建好会员权益，创建好会员卡，并在会员卡中添加会员权益。商城用户在“我的”页面可以看到开通会员卡入口，领卡后享受相应的权益。

另外，商城会员享受会员价格。在会员等级模式中，商家可以在会员等级处设置不同会员等级，在会员管理处给用户设置为相应的会员等级的会员，并给商品设置会员等级价格后，会员用户在商城就可以尊享会员等级价。在会员卡模式中，商家可以给商品设置不同的会员卡价，领取了相应会员卡的用户在商城尊享会员卡价格。

## C、会员卡权益

目前新增权益支持“领卡送积分”、“消费积分翻倍”以及自定义权益。

其中权益为“领卡送积分”的权益，是指当用户领取一张有此权益的会员卡领取成功时，系统自动给发放相应积分；

其中权益为“消费积分翻倍”的权益，是指当用户领取一张有此权益的会员卡之后，在有效期内条件下，用户去下单商品时，在原有的积分规则下发放给用户多倍积分；

其中权益为“自定义权益”的权益，是指由商家自行填写权益内容，自行给用户提供的服务权益。

用户如何获得会员卡权益？需要经过如下过程：

商家侧先新建权益，再新建会员卡，在会员卡编辑过程中关联建好的权益，最后启用会员卡；用户侧在登录商城后在“我的”界面，点击“开通会员卡”，进入会员中心界面，可以查看卡的权益，一级获得卡的条件，再进行领卡下单，完成领卡后即获得相应权益。具体如下：
