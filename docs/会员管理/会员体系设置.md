# 不同会员体系如何设置

## 1.会员等级模式

用户开通Apos系统后，系统默认设置的会员模式为：会员等级。用户可以在后台设置会员注册时需要的基础信息，包含手机号、姓名、邮箱、生日以及有效期等，可以根据需要设置这些信息是否必填。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fwt%40jieztech.com%2Fcustomer-manage.md%2Fefca66d5-3ed8-488e-83d4-081f8ffd6522_image.png?Expires=1804559838&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=cvRMiEGqqJoTCnB2fObpMbm01Gs%3D&x-oss-process=image%2Fformat%2Cpng)

商户后台的会员模式为会员等级时，菜单列表会出现‘会员等级’菜单，同时系统会自动创建一个‘基础等级’的会员等级。用户可以根据自己需要再创建新的会员等级。可以新增会员等级，新增时只需要设置会员名称以及每月限购金额。限购金额的作用是：会员可以以会员价或以会员价为基础享受的活动价购买的商品的金额，如果不设置则默认为不限。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fwt%40jieztech.com%2Fcustomer-manage.md%2F33bbb0d0-982b-4fa8-baa6-2efe266a7016_image.png?Expires=1804561138&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=L4pnL9KtnxD4YakLbJjUwuIaNgg%3D&x-oss-process=image%2Fformat%2Cpng)

会员等级对应的会员价可以在：商户后台-\>价格管理-\>基础零售价/会员零售价内进行设置

![](https://gvt-libs.oss-cn-hongkong.aliyuncs.com/Apos-Helper-Document/customer-management/f80ef27788002cae4522957e18fd4ccc.png)

会员等级的允许参加活动可以设置开启或者关闭，该设置的作用是：允许参加活动表示享受会员价的同时，还可以继续享受促销活动和整单折扣；不允许参加活动表示会员价为最终成交价。

![](https://gvt-libs.oss-cn-hongkong.aliyuncs.com/Apos-Helper-Document/customer-management/c523ae901723d03200d2dd0f7e62e96d.png)

会员人数是当前会员等级下的会员数量。

![](https://gvt-libs.oss-cn-hongkong.aliyuncs.com/Apos-Helper-Document/customer-management/0033e92d304271f88fe199b3eca9b9cb.png)

点击会员人数后，页面会跳转到会员管理页面，页面中会以点击的会员人数对应的会员等级为搜索条件，搜索出对应会员等级的会员信息。

![](https://gvt-libs.oss-cn-hongkong.aliyuncs.com/Apos-Helper-Document/customer-management/9009b9ab5d8db98d3e496f41d1937613.png)

会员等级可以进行编辑，点击对应会员等级的编辑按钮，可以对会员等级的名称和限购金额进行修改，修改后点击保存即可。

![](https://gvt-libs.oss-cn-hongkong.aliyuncs.com/Apos-Helper-Document/customer-management/3b414cc72b66d9fcdf172ac96b58b400.png)

会员等级状态控制当前会员等级是否可用，默认会员等级为启用状态，可用对会员等级进行启用和禁用。当会员等级被设置为默认等级后，无法对该会员等级进行禁用。

![](https://gvt-libs.oss-cn-hongkong.aliyuncs.com/Apos-Helper-Document/customer-management/c2d196bd0835b122b0f6b9d81578c725.png)

当会员等级列表内存在多个会员等级时，如果想删除某一个会员等级，点击对应会员等级的删除按钮，确认删除即可。如果当前会员等级设置为默认等级时，不能对该会员等级进行删除，当会员等级下已经存在了会员的情况下，也无法删除该会员等级。

![](https://gvt-libs.oss-cn-hongkong.aliyuncs.com/Apos-Helper-Document/customer-management/f5161b2d663d1adeff869b2d844423d6.png)

### 会员卡权益

目前新增权益支持“领卡送积分”、“消费积分翻倍”以及自定义权益。

其中权益为“领卡送积分”的权益，是指当用户领取一张有此权益的会员卡领取成功时，系统自动给发放相应积分；

其中权益为“消费积分翻倍”的权益，是指当用户领取一张有此权益的会员卡之后，在有效期内条件下，用户去下单商品时，在原有的积分规则下发放给用户多倍积分；

其中权益为“自定义权益”的权益，是指由商家自行填写权益内容，自行给用户提供的服务权益。

用户如何获得会员卡权益？需要经过如下过程：

商家侧先新建权益，再新建会员卡，在会员卡编辑过程中关联建好的权益，最后启用会员卡；用户侧在登录商城后在“我的”界面，点击“开通会员卡”，进入会员中心界面，可以查看卡的权益，一级获得卡的条件，再进行领卡下单，完成领卡后即获得相应权益。具体如下：

领卡送积分:领取此卡立即送n积分。
消费积分翻倍：（1）允许设置单次使用。（2）允许设置多次使用，例N天之内，可以使用M次。（3）允许设置不限次数使用
