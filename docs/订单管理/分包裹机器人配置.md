# 分包裹机器人配置

按供应商分包裹机器人
====================

基本介绍：
----------

按供应商分包裹机器人，这个功能目前仅针对聚水潭保税业务，需开通聚水潭应用。聚水潭业务中，一个商户下可能有来自多个聚水潭供应商的商品，如果用户在商城一次同时将来自多个聚水潭供应商的商品一起下单，这些商品将按照聚水潭供应商来源分在不同的包裹。

【操作指引】：

1、在APOS管理后台，找到系统个管理菜单，再点击“自动化机器人”菜单，新建一个机器人
