# 历史订单导入


## 基本介绍

“订单导入”功能包括订单导入、订单详情、H5/微信小程序中展示历史订单.

## 使用介绍

使用导入的方式将历史订单数据导入，以订单编号为判断导入的历史订单数据是否是重复或新新订单数据。已经成功导入历史订单数据可以查看订单的详情信息。商城中进入订单列表—全部订单—滑动至底部，点击查看历史订单按钮&gt;即可查看导入的历史订单数据。
