# 如何查看一笔订单是直接分销还是独立分销的订单

无论是直接分销还是独立分销业务的订单，无论是否有佣金收益，该笔订单会记录在相应业务佣金管理里面。

可以拿订单号到APOS管理后台--直接分销--佣金管理或者 APOS管理后台--独立分销--佣金管理用订单号条件查询。查询得到，则是相应业务的订单，如果都查不到，则既不是直接分销也不是独立分销的订单。
