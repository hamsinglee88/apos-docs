# 如何设置订单签收一段时间后不可再申请退单


## 操作步骤：

## 一、登录APOS管理后台账号

## 二、找到左侧菜单栏的“系统管理”--“系统设置”

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E5%A6%82%E4%BD%95%E8%AE%BE%E7%BD%AE%E8%AE%A2%E5%8D%95%E7%AD%BE%E6%94%B6%E4%B8%80%E6%AE%B5%E6%97%B6%E9%97%B4%E5%90%8E%E4%B8%8D%E5%8F%AF%E5%86%8D%E7%94%B3%E8%AF%B7%E9%80%80%E5%8D%95.md%2F3772544f-55c0-4fa4-b4ad-41ec908bfd01_image.png?Expires=1824517760&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=PVRCaBp5i89nPQsUBx5OV3HZgbw%3D&x-oss-process=image%2Fformat%2Cpng)

## 三、找到设置项“收货后可申请售后时间”

配置需要的时间即可。用户订单签收达到一定时间后，商城不再展示申请售后按钮
