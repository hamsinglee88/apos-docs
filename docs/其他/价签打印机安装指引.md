# 价签打印机安装指引


Apos的商户后台提供纸质价签打印功能，需要配套的价签打印机和热敏打印纸；

价签打印的系统功能介绍请参考操作指引的《Apos商品管理使用指引》，这里只介绍打印机的安装配置：

1.先安装好打印机设备

![](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fcodyxiong%40jieztech.com%2F%E6%A0%87%E7%AD%BE%E6%89%93%E5%8D%B0%E6%93%8D%E4%BD%9C%E6%96%87%E6%A1%A3.docx%2F595c7f0c-f448-492a-8dec-35fa672cf6a0_image1.png?Expires=1796463302&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=dV%2Br%2F6KnUEUsIYwNzt5J%2B1BE8fw%3D&x-oss-process=image%2Fformat%2Cpng)

1. 在电脑上安装打印机驱动，下面是安装包（鼠标右键打开超链接）

   [<u>\[金山文档\]
   GP-3200TUBdyjqd.zip</u>](https://kdocs.cn/l/cfiX4MrogdlP)
3. 安装好之后可以打印测试页，测试是否打印正常

   ![](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fcodyxiong%40jieztech.com%2F%E6%A0%87%E7%AD%BE%E6%89%93%E5%8D%B0%E6%93%8D%E4%BD%9C%E6%96%87%E6%A1%A3.docx%2F134701ad-8cc4-4410-9d39-fd046e23a3e6_image2.png?Expires=1796463302&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=adCgldcj5Ps00CqdagOqt1C5yL0%3D&x-oss-process=image%2Fformat%2Cpng)
