## 销售分析


### 销售分析

销售分析主要是用来查询销售数据，未设定销售目标，也能正常查询出实际发生的销售数据。

每个字段详解：

订单数量：销售订单数量，不减退货单数量；

日销售额：当日商品价(AU/CN为含税价，US为不含税价）实际销售额（减退货金额），含CRV Fee，不含物流；

销售目标：设定的当日的销售目标，点击修改图标可再次修改当前门店当日销售目标(操作同销售目标中设置单日的销售目标)；未设定销售目标值，销售目标、销售完成率、销售差额、销售偏差比显示为“-”；

销售完成率(%)：日销售额 / 销售目标；

销售差额：日销售额 - 销售目标；

销售偏差比(%)：销售差额 / 销售目标；

去年销售额：去年同日的商品价(AU/CN为含税价，US为不含税价）实际销售额（减退货金额），含CRV Fee，不含物流；去年销售额为0时，去年销售额、销售对比显示为“-”；

销售对比(%)：日销售额 / 去年销售额；

平均销售：日销售额 / 销售订单数

### 销售目标

销售目标用于查询和设置销售目标数据，未设定销售目标，能正常查询出记录。可以进行单独设置与批量设置。批量支持对单个门店的多个日期进行设置销售目标。

各字段详解：

通过自定义手工输入销售目标或批量设置销售目标，未对标历史销售，参考日期、参考日销售额、增长率(%)显示为”-”；

通过参考考历史销量设置销售目标，参考日期为设置时选择的的参考日期；

参考日销售额：参考日的商品价(AU/CN为含税价，US为不含税价）实际销售额（减退货金额），含CRV Fee，不含物流；销售额为0时为“-”；

销售目标：运营人员设计的销售目标；

增长率(%)：销售目标 / 参考日销售额 - 1，无参考日或参考日销售额为0时，增长率(%)显示为“-”，增长率计算出来为负数，以红色字体显示；

### 销售对比

销售对比用于对比分析历史销量，销售日期默认为过去7天(不包括当天)。

各字段详解：

用于对比分析历史销量，销售日期默认为过去7天(不包括当天)；

对比方式包括同比、环比和自宝义，默认环比

设销售日期为A~B,对比日期为X~Y,则销售日期和对比日期的关系为：

同比：对比去年同期（对比日期不可修改），X = A – 1年，Y = X + (Y – X)，如销售日期为“2019-01-01 ~ 2019-01-31”，则对比日期为“2018-01-01 ~ 2018-01-31”；

环比：对比上一期（对比日期不可修改），Y = A – 1天，X = Y – (Y - X)，如销售日期为“2019-01-01 ~ 2019-01-10”，则对比日期为“2018-12-22 ~ 2018-12-31”；

自定义：用户自己选择对比日期，对比日期的天数比例与销售日期的天数是相同的，否则点击查询时提示“对比日期与销售日期的天数不一致!”，修改起始对比日期计算结束日期，Y = X + (B –A)，修改结束对比日期计算开始日期，X = Y – (B – A);

l对比结果为，销售日期范围的日期与对比日期范围的日期一一对比；

l增长额 = 销售日期日销售额 – 对比日期日销售额

l增长率 = 增长额 / 对比日期日销售额
