# 特价活动使用手册

版本：V 1.0.0

## 一、开启特价活动

如果想在商城内使用特价活动，点击营销活动--->特价活动--->新建活动

## 二、设置特价活动

### 1、创建特价活动

名词解释：
活动单位：选择参与活动的门店
活动时间：设置固定的活动开始时间与结束时间
适用人群：可以进行选择哪些用户可以参与此特价活动
限购类型：用户购买特价商品时，进行限购
允许折上折： 用户购买特价商品时，允许享受多个优惠

1、选择活动单位，可以同时选择多个门店参与特价活动

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FSpecial-offers.md%2Ff7da1fff-f683-457e-a280-9a08b0e46392_image.png?Expires=1805879214&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=L3%2B%2F%2FBSvQfYO9pZQL2xCJUHSxxo%3D&x-oss-process=image%2Fformat%2Cpng)

2、点击输入活动名称，活动时间、活动内容
![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FSpecial-offers.md%2F820786f2-69b8-4c54-a6d0-1baaaa9c9284_image.png?Expires=1805879237&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=UkYhPS3NjA22tyDluYlFydrWPEU%3D&x-oss-process=image%2Fformat%2Cpng)
3、选择适用人群，点击按钮即可
所用人：所有人都可以参与此特价活动
部分用户：根据用户标签选择适用人群,新会员（注册后下单数量为0的用户）为系统自带的标签，其它标签需要自建。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FSpecial-offers.md%2Fbc650a4b-f78f-4707-8e99-bd10979ae67c_image.png?Expires=1815364318&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=joDgHaaSVnDgzl7kdSOl8FGvIOs%3D&x-oss-process=image%2Fformat%2Cpng)
4、选择限购类型
不限购：购买特价商品时，可以无限下单
每单限购：选择每单限购时，需要输入限购数量框中输入数量，购买特价商品时，每单限制购买数量
![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FSpecial-offers.md%2Fd4f777e9-9d2b-49a8-a89c-2d9afc5e612d_image.png?Expires=1805879297&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=rlwnSWP1LNWtKFxapuGhstGekLc%3D&x-oss-process=image%2Fformat%2Cpng)

5、选择允许折上折
允许：可以参与多种优惠活动
不允许：不可以参与多种优惠活动
![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FSpecial-offers.md%2F8502b8da-3d5a-4686-8f57-06d4c1fc0671_image.png?Expires=1805879325&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=dP%2FDWL5RhU1iUw3cZrMbSXXEl30%3D&x-oss-process=image%2Fformat%2Cpng)
6、添加特价商品
添加商品：单独进行选择商品，享受特价
导入商品：用模板的方式，将商品进行导入
折扣率（%）：输入折扣率，点击批量设置按钮，此活动所有参与特价的商品享受折扣率
![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FSpecial-offers.md%2Fdd83aa33-8f43-4c4d-b5de-7bf3de09f2e4_image.png?Expires=1805879355&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=Fcet3eNJwRC2MnfwqiEBJ3Aaoy8%3D&x-oss-process=image%2Fformat%2Cpng)
