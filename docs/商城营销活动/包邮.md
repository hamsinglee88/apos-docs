# 一、包邮活动


## 一、使用包邮活动

如果想在商城内包邮活动，需要在营销活动--包邮活动

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FPackage-mail-activity.md%2Fb519a48e-d1a2-45eb-8009-b295b6bb49c1_image.png?Expires=1807345514&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=9SY8Lxpube7utxQ%2F0qFP5qQBMAk%3D&x-oss-process=image%2Fformat%2Cpng)

创建一个新的包邮活动
![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FPackage-mail-activity.md%2F22fd2627-ac65-411b-8460-3a7f4df6a070_image.png?Expires=1807345555&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=6K0wQiKxIWNU80CJ1A0f%2Bc%2FzfnE%3D&x-oss-process=image%2Fformat%2Cpng)

选择参与活动的单位（点击显示下拉框，下拉框中门店以及仓）

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FPackage-mail-activity.md%2F9d69740f-4808-43b2-afe3-ee31dfc760f9_image.png?Expires=1807345616&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=ch3cKr%2FBRLuC8lBTQsuqxPIIAUM%3D&x-oss-process=image%2Fformat%2Cpng)

输入包邮活动的活动名称（最多可以输入100个字符）

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FPackage-mail-activity.md%2F4f0949a0-3fb0-44dc-a649-61a8e8df7dde_image.png?Expires=1807345639&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=JhPZCbV1AyJ0x0CzD8V6sY6x798%3D&x-oss-process=image%2Fformat%2Cpng)

选择活动范围（分为线上商城，线下门店）可以进行多选

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FPackage-mail-activity.md%2F237f782c-f63f-4786-8d32-c5366397cdd0_image.png?Expires=1807345666&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=i2qcE1FTafKPbgj%2BzKYzpZWWins%3D&x-oss-process=image%2Fformat%2Cpng)

输入活动有效时间
长期有效：活动时间一直生效
固定时间：输入有效的活动开始时间与结束时间，在这个时间内活动有效

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FPackage-mail-activity.md%2Fc925f8a1-e1df-4bc9-91a5-d3b5cdec110c_image.png?Expires=1807345693&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=H2ylW93lTabgpbryxvOOaltmVeQ%3D&x-oss-process=image%2Fformat%2Cpng)

选择分箱规则
不指定：所有的物流都可以进行分箱
指定：选择指定的物流进行分箱

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FPackage-mail-activity.md%2F5b9cba57-23f8-4adb-a602-4faf1db58b9d_image.png?Expires=1807345723&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=4iS46%2FMigxE4vKAcYYKtUvgmrjg%3D&x-oss-process=image%2Fformat%2Cpng)

选择指定包邮地区
不指定：所有的地址都可以进行包邮
指定：点击添加包邮地区按钮，弹出选择包邮地区弹窗，选择其中地址可以进行包邮

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FPackage-mail-activity.md%2F1ba07920-3a7c-4638-9a95-c03ca747433a_image.png?Expires=1807345752&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=EBdL5FtxXTMkxTvIiQewHVx7D%2FI%3D&x-oss-process=image%2Fformat%2Cpng)

选择活动类型
满金额包邮：达到活动门槛中购买金额即可包邮
满数量包邮：达到活动门槛中购买数量即可包邮
![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FPackage-mail-activity.md%2F939dd18e-9ff1-47f2-896d-f19e031e8bf6_image.png?Expires=1807345821&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=Xe0Wbjy6t5Ech355qscb39454p8%3D&x-oss-process=image%2Fformat%2Cpng)

选择参与包邮的商品
全部商品：所有商品
全部本地品：所有一般贸易商品参与包邮
全部跨境品：所有保税商品参与包邮
![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FPackage-mail-activity.md%2F8d71cb6d-7419-45b5-adb6-86e785faf2d5_image.png?Expires=1807345903&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=CPonVh8Xgilt3Zqq9GtjDpuRAq8%3D&x-oss-process=image%2Fformat%2Cpng)
