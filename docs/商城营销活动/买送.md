# 一、买送活动

商家可以在营销活动中创建买送活动，设置买几件送几件，买送的商品分为促销品和赠品，其中赠品可以设置为免费，或者也可以设置赠品是打几折,支持多级阶梯，最多支持五个阶梯

其中，如果设置了多级买送活动，如果用户添加到购物车的促销商品达到高阶梯的门槛条件，那么这个用户的购物车商品只会匹配到这个一个等级的活动，不再匹配这个活动的其他等级。

设置好启用买送活动后，在商城该商品会有“买送”标签展示

如果后台设置的买送的赠品只有一个商品，则用户添加促销品到购物车达到门槛时，系统自动带上赠品添加到购物车；

如果后台设置的买送的赠品不止一个商品，则用户添加促销品到购物车后，需要再从促销品的更多标签“&gt;”进入赠品列表页面添加赠品到购物车

具体匹配规格举例：


活动1：促销商品A 赠品B   等级一买2送1   等级二买3送2

购物车商品：

情景1：Ax2  Bx1 匹配等级一

情景2：Ax3 Bx1 匹配等级一

情景3：Ax3  Bx2 匹配等级二

情景4：Ax4 Bx2 匹配等级二

情景5：Ax5 Bx3 匹配等级二

情景6：Ax6 Bx4 匹配等级二 2次
