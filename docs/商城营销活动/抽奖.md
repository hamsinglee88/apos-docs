# 抽奖活动

> 文档地址：商城营销活动/抽奖.md


## 基本介绍

商家可以在应用商店---抽奖活动应用创建活动。目前抽奖活动仅商家的主商城可以参加活动，子店不可以参加抽奖活动。设置奖品，可以将商品（仅限一般贸易品）、代金券和积分设置为奖品。

## 一.开启抽奖活动应用

如果想在商城内使用团购，需要在应用商店内开通团购应用，开通入口在系统页面右上角->应用商店->商店首页内找到抽奖活动

## 二.创建抽奖活动，设置活动规则

1.点击创建活动

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Flottery_draw.md%2F537f3e46-6c24-4feb-9edc-735330558d00_image.png?Expires=1805855269&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=YkogFZ7BcvYaF6%2FUn9tHUsl%2FZsE%3D&x-oss-process=image%2Fformat%2Cpng)

2.填写活动基本信息，包括活动名称，活动时间，活动规则

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Flottery_draw.md%2Ff8b47d6d-6d8e-4c9d-a9a6-062bf8786358_image.png?Expires=1805855306&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=gcXuhlEM8oh13qm7Dd0uq6iiUv8%3D&x-oss-process=image%2Fformat%2Cpng)

3.添加奖品，分别可以添加代金券，添加积分，添加商品（仅限一般贸易商品）作为奖品

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Flottery_draw.md%2F9a772098-f083-459c-a16d-3b46b93360ac_image.png?Expires=1805855625&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=d41oiDimXl56uFc0ckK4K7%2BZjs4%3D&x-oss-process=image%2Fformat%2Cpng)

4.设置奖品发放数量---上图中，设置发放数量N份，则中奖将被限制最多可中奖N份，中奖完毕后，再抽奖将不会再中该奖品

5.设置奖品概率。设置8个转盘各自被转到的概率之和要等于100%

![1648176334(1).png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Flottery_draw.md%2F7d9f16a6-d96d-4eef-900d-9dcfe1606a2f_1648176334%281%29.png?Expires=1805856425&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=v2LurRlPAmkbv07V3sLEKMhjGZw%3D&x-oss-process=image%2Fformat%2Cpng)

6.设置抽奖次数。其中可设置没人每天抽奖次数（免费次数），每人一共抽奖次数（免费次数）。额外抽奖次数--在下单达到设定的条件后，系统自动发放额外抽奖次数。另外可设置每人中奖次数，如果某用户用户抽奖达到了中奖次数，将不再会中奖。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Flottery_draw.md%2F541acd11-b892-4b90-936e-4c16e5f11dd8_image.png?Expires=1805856645&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=3y1FOjsomwCN5AFfzvcxZ5JTmYY%3D&x-oss-process=image%2Fformat%2Cpng)

最后记得点击保存活动

## 四.兑奖

用户中奖后可以点击立即使用或者，继续抽奖，后续也可以再点击抽奖界面左上角的“奖品”进入活动奖品列表页，查看奖品；

其中，如果中奖代金券或者积分，则系统直接发到用户账号上，在我的页面可以去查看，代金券，或者积分增多。如果中奖的是商品，则需要用户在抽奖活动页面，对商品奖品点击使用，去下单的方式兑换奖品

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Flottery_draw.md%2F79b05a97-4cda-4355-826a-cc99a53a863d_image.png?Expires=1805858333&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=62gpG0ncOXpH2QMG4y4H67joDKc%3D&x-oss-process=image%2Fformat%2Cpng)

如果是对代金券奖品点击立即使用，会跳转到商城首页，用户选择商品下单用券；

如果是对积分奖品点击立即使用，会跳转到积分商城，下单积分商品；

如果是对商品奖品点击立即使用，会直接去下单，


其中注意，商品是仅限（一般贸易商品），商品奖品在兑换是，是0元下单，免除运费。

商品奖品下单时，商家必须得在--商户后台--门店管理--支付管理处，开启适用商城的余额支付方式

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Flottery_draw.md%2F25a5c0bb-67b3-4d78-be0e-9f838fa4fed4_image.png?Expires=1805859410&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=12UuWNuHQC0HEd%2B24BAp3xmDdwc%3D&x-oss-process=image%2Fformat%2Cpng)

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2Flottery_draw.md%2F7d9c761a-16d7-4b47-98ae-df250ffd3cce_image.png?Expires=1805859455&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=oPKSDh4oIw%2Fgp98SLOI3eNMzSnQ%3D&x-oss-process=image%2Fformat%2Cpng)

（奖品下单，一定要开启余额支付方式，0元单不可以走在线的微信支付等方式）

另外，奖品下单不支持退款（0元单不支持退款）
