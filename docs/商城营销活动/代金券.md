# 一、代金券活动


代金券可以用来在商城下单时抵扣商品金额，不可以抵扣运费、税费。

让用户使用代金券的步骤如下：

## 【领券中心】

步骤1:创建代金券，在创建代金券时，同样选择商城绑定的门店作为适用门店，正确填写代金券的各项信息保存后启用

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FVoucher-activities.md%2F168966a9-a5d6-4a80-afd9-66871e556dfe_image.png?Expires=1823048784&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=WUJl7fugdDlzaeC0FaI0ttegF0g%3D&x-oss-process=image%2Fformat%2Cpng)

启用

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FVoucher-activities.md%2Ff974c68d-4aef-45bd-98bd-68838fb5ac15_image.png?Expires=1808107212&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=m5SzL6hJVD8Bf73t1uug8h8Crvk%3D&x-oss-process=image%2Fformat%2Cpng)

步骤2：打开商城代金券活动，让用户获得券。让用户获得代金券有以下几种方式：1）注册送券，商户在管理后台创建一个注册赠送类型的商城代金券活动，在活动下添加这张券，然后新用户在商城注册就会赠送相应的券；2）用户领取，创建一个用户领券的代金券活动，用户可以在商城的领券中心看到该活动，在活动内容中领券；3）消费赠送，创建一个消费送券活动，用户在商城消费达到活动赠送券的金额就赠送相应的券；

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FVoucher-activities.md%2F0ac3342e-bc90-424e-9c34-9bd592107a81_image.png?Expires=1808107420&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=xgpoUO4hzIi1yBJofCWOoldwg0g%3D&x-oss-process=image%2Fformat%2Cpng)

添加优惠券--关联第一步创建的代金券--保存--启用

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FVoucher-activities.md%2Fe657f042-5aa6-48b2-80b8-49d43b1b41ed_image.png?Expires=1808107317&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=Xig1otmaqM3ti6CIMMpXk2D%2FU6E%3D&x-oss-process=image%2Fformat%2Cpng)

客户即可在商城--领券中心--领取

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2FVoucher-activities.md%2F4fa5ee6c-793f-48b3-aec1-cad43c05fd25_image.png?Expires=1816595324&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=rB4hS1DgoPgsNJTDQT%2Bfel0AHSo%3D&x-oss-process=image%2Fformat%2Cpng)

步骤3：返回代金券，商家主动发券给用户，在代金券列表对券进行发券操作，可以不指定客户，对整个商城的用户全体发券，也可以指定用户，只对商城的部分用户发券；

## 【礼品券】

【礼品券】用于商户制作线下券，在线下发给用户，用户使用券码和密码在商城“我的”--“礼品券”菜单可以进行绑定代金券

操作步骤同上，区别在于【发券】的时候，会员类型选择“否”--生成--导出excel表格，导出的报表有券码和密码
