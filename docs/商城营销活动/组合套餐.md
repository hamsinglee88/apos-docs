# 组合套餐

> 文档地址：商城营销活动/组合套餐.md


在营销活动中创建套餐活动，将几个商品添加到一个套餐，设置一个套餐价，用户在商城浏览到套餐，可以看到原价与优惠后的套餐价，整体加入购物车、下单。
