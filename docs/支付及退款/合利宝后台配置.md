# 合利宝支付操作说明


## 一、合利宝支付（线上）

### 1、配置合利宝线上支付

配置合利宝线上支付与配置其他第三方支付方式一样，在商户后台-门店管理--支付管理页面新增支付方式

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fkellywang%40jieztech.com%2Fhelipay.md%2F9b1d25ea-b09f-4c09-9d04-d39888f2aa63_image.png?Expires=1809832367&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=LNsiP7Xnus9Iyk2iTz3ZCg5zF%2Fc%3D&x-oss-process=image%2Fformat%2Cpng)

### 2、合利宝支付申报配置

合利宝线上支付可以支持支付单申报。如果需要开通合利宝支付报关，需要在应用商店中开动‘支付单申报’应用。在应用中开启合利宝支付报关，并配置相应的报关信息。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fkellywang%40jieztech.com%2Fhelipay.md%2F229aa942-99ed-48a6-9aa4-b2365b621b13_image.png?Expires=1808538935&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=oyvG1l5qPpmp9W%2FaL4OT4vEbpFQ%3D&x-oss-process=image%2Fformat%2Cpng)
