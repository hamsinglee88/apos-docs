# **科普：** **什么是商户API证书？如何获取商户API证书？**

功能介绍

API证书，是指由商户申请的，用来证实商户身份的证书。API证书由证书授权机构Certificate Authority（简称CA）颁发。证书中包含商户的商户号、公司名称、公钥等信息。请妥善保管API证书文件，防止泄露。

使用场景

APIv2 中，调用微信支付安全级别较高的接口（如：退款、企业红包、企业付款）

APIv3 中，调用微信支付所有接口

获取API证书方法（通过商户平台下载）

1、 （需要商户号的超级管理员才能操作）操作路径：

登录【微信支付商户平台 - 账户中心 - 账户设置 - API安全 - 申请API证书】申请证书，确定后请勿关闭页面

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E5%BE%AE%E4%BF%A1.md%2F5a67e787-787c-4bad-8479-5b58f6b29152_image.png?Expires=1816660335&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=kkQQCrYqnF8uke5ecg%2Fe0AZ1mBI%3D&x-oss-process=image%2Fformat%2Cpng)

2、点击下载证书工具；下载后，双击“WXCertUtil.exe”文件，选择安装路径后，点击申请证书

也可通过以下链接下载证书工具：

windows版本 ：https://wx.gtimg.com/mch/files/WXCertUtil.exe

mac版本 ：[https://wx.gtimg.com/mch/files/WXCertUtil.dmg](https://wx.gtimg.com/mch/files/WXCertUtil.dmg)
