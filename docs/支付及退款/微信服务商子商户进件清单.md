# **微信服务商子商户进件清单**

账号合规进件需要用到主体信息、经营信息、行业资质、结算账户、微信超级管理员五个部分材料

**进件必需材料清单汇总；需要客户准备提供**

1，营业执照照片，请上传彩色照片 or 彩色扫描件 or 加盖公章鲜章的复印件，可添加“微信支付”相关水印（如微信支付认证）

2，法人身份证正反面照片

3，支付成功页面显示的商家名称

4，客服电话

5，经营场景列出；业务场景所需要要的appid和应用截图

6，所属行业划分

7，开户行信息（开户名称、开户行、开户支行、基本银行账号）

8，管理员信息（姓名、身份证号、手机号、邮箱）

**第二部分；经营信息**

需要材料清单；

1、在支付完成页向买家展示，需与微信经营类目相关。

2、简称要求：
① 不支持单纯以人名来命名，若为个体户经营，可用“个体户+经营者名称”或“经营者名称+业务”命名，如“个体户张三”或“张三餐饮店”；
② 不支持无实际意义的文案，如“XX特约商户”、“800”、“XX客服电话XXX”；

3、客服电话；

4、经营场景描述；
