# 余额支付


余额以用户为单位，余额支付是一种支付方式，已充值方式存在一定的金额，进行下单支付。

当用户存在余额时，商品下单支付会显示余额支付方式，可以选择不同的支付方式进行支付。用户购买“积分商品”时，需要开启余额支付方可支付

当用户没有余额时，商品下单支付不会显示该余额支付方式，
