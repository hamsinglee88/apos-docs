# 搭建微信小程序商城

## 一、前提条件

已经开通了APOS商户端账号，开通商城

## 二、小程序添加开发人员

联系APOS系统对接人员，联系将开发人员微信添加至小程序的项目成员，此项在系统对接群联系即可

### 三、提供微信小程序秘钥给开发人员

创建新的小程序后，可到如下菜单生成小程序秘钥：


1. 登录小程序管理后台
2. 找到“开发”--“开发管理”--“开发设置”

   ![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E6%90%AD%E5%BB%BA%E5%BE%AE%E4%BF%A1%E5%B0%8F%E7%A8%8B%E5%BA%8F%E5%95%86%E5%9F%8E.md%2F6002da27-95ee-47a6-9e3a-742a14a69681_image.png?Expires=1824363926&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=Fa80PBQUm7vEqD7tcyeupTv%2Blcg%3D&x-oss-process=image%2Fformat%2Cpng)

   ## 四、开发人员配置小程序

   （此步无需商家操作）

   ## 五、提交微信审核

   1、登录微信小程序

   2、找到菜单“管理”--“版本管理”--“开发版本”，找到最新上传的版本提交审核![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E6%90%AD%E5%BB%BA%E5%BE%AE%E4%BF%A1%E5%B0%8F%E7%A8%8B%E5%BA%8F%E5%95%86%E5%9F%8E.md%2F4ced3bce-db67-4445-9bdd-f1debdd278b8_image.png?Expires=1824364371&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=uzep92KvIFXA0LHEMHqgw2WCvuU%3D&x-oss-process=image%2Fformat%2Cpng)

## 六、发布版本

待微信官方审核通过，在来版本管理，点击发布即可
