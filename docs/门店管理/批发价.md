# 批发价


修改批发价，生成调价单和调商品批发价报表记录

## 批发价列表

进入后台，点击“商品管理”，然后点击“批发价”，进入列表页；

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fkenfeng%40jieztech.com%2Fcommodity-management.md%2F7a83ec79-73b3-4d8e-8276-9eca181362fd_image.png?Expires=1794298945&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=nGfU4IcBuZcMOlBa5e3ahRo5%2FEo%3D&x-oss-process=image%2Fformat%2Cpng)

## 修改批发价

1. 列表进行批发价的设置
2. 调价单列表，产生调批发价的调价单

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fkenfeng%40jieztech.com%2Fcommodity-management.md%2Fe39d1355-3115-46b4-a5cf-2a470929b01b_image.png?Expires=1794299043&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=2sbO7ak5TwfzgMr9RvoC%2BAdBow0%3D&x-oss-process=image%2Fformat%2Cpng)

## 导入

在批发价列表界面，点击导入，下载模板，输入批发价信息后，选择该文件，确定上传，就可以完成批发价的修改

如果不想手动输入商品等信息，可通过先导出再在导出的文件中进行批发价设置，保存excel后，再上传该文件即可---推荐使用

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fkenfeng%40jieztech.com%2Fcommodity-management.md%2Fcb9f0739-c9be-47e1-afff-dc344b9fe7c4_image.png?Expires=1794299221&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=AAyjjvnvgbmd55bsIcHn%2FxSNSlQ%3D&x-oss-process=image%2Fformat%2Cpng)

## 导出

选择门店，查询到要导出的数据，点击“导出”，完成批发价的导出功能，文件名默认为“批发价.xls”

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fkenfeng%40jieztech.com%2Fcommodity-management.md%2F70becfa4-a153-41d1-a2d4-c420d8db3b57_image.png?Expires=1794299566&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=HyJFY9xEQJ%2B6EBCu6EtpdFcQaMo%3D&x-oss-process=image%2Fformat%2Cpng)
