# 调价单

进入后台，点击“商品管理”，然后点击“调价单”，进入列表页；

调价单支持 调基础零售价，调门店零售价，调批发价。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fkenfeng%40jieztech.com%2Fcommodity-management.md%2F4b1c0a23-d602-40f1-b2c5-009f25e620ac_image.png?Expires=1794299664&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=tVuJ8WARBsvtEAgRVY6NGlNo1O4%3D&x-oss-process=image%2Fformat%2Cpng)

1. 商品信息修改价格，批量修改价格，此处会产生调基础零售价的记录
2. 门店价格修改价格，批量调价，此处会产生调门店零售价的记录
3. 批发价修改价格，此处会产生调批发价的记录
4. 此处可手动创建定时生效的调价单，比如基础零售价，门店零售价，批发价，创建并确认该调价单后，到了对应的生效时间，价格更新
