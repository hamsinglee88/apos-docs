# 快捷菜单

> 文档地址：门店管理/快捷菜单.md


快捷菜单功能仅支持win版收银客户端，商户可以把一些常卖的商品或者物流添加到收银机的首页，收银员可以不用扫码，直接手动选择；

支持分类，并在分类下添加商品；

支持排序，自定义显示的顺序；

支持启用、禁用、和删除；

快捷菜单的新增和修改完成后，需要在收银机端点击“数据同步”才能生效；
