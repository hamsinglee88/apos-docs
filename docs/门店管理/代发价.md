# 代发价


修改代发价，生成调价单和调商品代发价报表记录

## 代发价列表

进入apos后台，点击“价格管理”--\>“代发价”，进入代发价列表界面。

![](https://gvt-libs.oss-cn-hongkong.aliyuncs.com/Apos-Helper-Document/price-management/f6cb05b913793cd802c4705dca166174.png)

修改代发价，并保存

- 选择对应门店的商品修改商品的代发价和等级代发价，点击保存

![](https://gvt-libs.oss-cn-hongkong.aliyuncs.com/Apos-Helper-Document/price-management/9be383b5527ea1214c7ac98932fa7073.png)

- 在调价单列表，查看修改代发价生成的调价单

![](https://gvt-libs.oss-cn-hongkong.aliyuncs.com/Apos-Helper-Document/price-management/475004b69cab676ec786d09c9f3d869c.png)

- 在报表中心--\>商品调价报表，查看调价报表中调代发价报表信息

![](https://gvt-libs.oss-cn-hongkong.aliyuncs.com/Apos-Helper-Document/price-management/a4d07f17eeef9b84b923be74959db44f.png)

## 导出

查询到要导出的数据，点击“导出”，完成代发价的导出功能，文件名默认为“代发价.xls”

![](https://gvt-libs.oss-cn-hongkong.aliyuncs.com/Apos-Helper-Document/price-management/73c9bc6dccdaa2a627cb5d7961fac4d0.png)

## 导入

门店零售价界面，点击“导入”--\>“下载模板”--\>设置价格信息并保存--\>“选择文件”--\>“上传”，完成价格的批量设置。

![](https://gvt-libs.oss-cn-hongkong.aliyuncs.com/Apos-Helper-Document/price-management/353f9ff57069a354ca98449a5b478818.png)
