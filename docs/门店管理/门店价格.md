# 门店价格


门店价格是用来管理各门店商品的销售状态，上下架，和销售价格，默认所有商品为在售状态，且价格为商户在商品信息设置的基础零售价，可以通过“价格类型”更改为“门店价”，来设置门店自己的价格；

销售状态：销售状态有正常、清货、停售三种状态

正常：商品可进可出，可正常零售和出入库业务；

清货：商品能出不能进，可正常零售，不能采购/补货入库；

停售：商品既不能出也不能入，零售和出入库业务都限制；

进入后台，点击“商品管理”，然后点击“门店价格”，进入列表页；

## 修改商品的状态

选择某个sku修改销售状态，点击保存，成功修改商品的销售状态（备注：销售状态为正常和清货，都会在客户端进行展示，用户可以购买）

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fkenfeng%40jieztech.com%2Fcommodity-management.md%2F6fe0ea43-9ee8-490d-ace0-472236354850_image.png?Expires=1794293147&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=vaqOOYSgjabEF6RY8enlrOrdAv8%3D&x-oss-process=image%2Fformat%2Cpng)

## 修改价格类型

如果某sku的价格类型选择了基础价，则修改商品信息的基础零售价，这里的价格相应更改；

如果某sku的价格类型选择了门店价，则修改基础零售价不影响该sku的价格，其他选择基础价的，则更新价格；

成功修改价格并保存，在“调价单”界面新增一条门店调价单记录。

## 导出

选择门店，查询到要导出的数据，点击“导出”，完成门店价格的导出功能，文件名默认为“门店价格.xls”

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fkenfeng%40jieztech.com%2Fcommodity-management.md%2F2d64f14f-61f5-40c7-9c78-b3d2fa1113dd_image.png?Expires=1794293622&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=pigbW4gWbrMPqQRdXZxu8aJIt0Y%3D&x-oss-process=image%2Fformat%2Cpng)

## 导入

门店价格界面，点击“导入”-->“下载模板”-->设置信息并保存-->“选择文件”-->“上传”，完成设置。

1. 导入时，价格类型为基础价，则更改价格，该sku的价格不更新，保持和基础价一致
2. 导入时，价格类型为门店价，则更改价格，该sku的价格更新
3. 导入时，修改sku的上下架状态，是否允许折扣，销售状态等，可以成功设置

   如果不想手动输入商品等信息，可通过先导出门店价格信息，再在导出的文件中进行设置，保存excel后，再上传该文件即可---推荐使用

   ![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fkenfeng%40jieztech.com%2Fcommodity-management.md%2Fc8eaa6f7-9dbf-4113-aaa9-c60330a15ddc_image.png?Expires=1794293709&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=ZZMCAzMXhJzEd3%2BdUX7lfstu4W4%3D&x-oss-process=image%2Fformat%2Cpng)

## 批量调价

在门店价格界面，点击“批量调价”，进入批量调价界面。

调价成功后，调价单列表增加一条调价记录。
