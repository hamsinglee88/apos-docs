# 微信小程序商城跳转H5

## 说明：

微信小程序跳转到H5，需要在微信小程序配置业务域名，配置业务域名需要上传微信提供的随机码上传到H5域名的服务器，因此目前APOS商城小程序跳转H5仅支持跳转到APOS商城的H5页面。

## 操作步骤：

## 一、配置小程序业务域名

联系APOS的开发人员配置小程序业务域名

## 二、装修小程序跳转H5

登录APOS商户后台，进到商城装修页面，添加任意一个组件，选择跳转链接，选择小程序跳转H5即可
