# 商城如何做折扣活动？

> 文档地址：常见问题FQA/商城如何做折扣活动.md


目前商城 支持的折扣活动有金额满折，即满多少金额后打多少折，还有件数满折，即购物车商品达到一定件数条件后，可以打多少折。
