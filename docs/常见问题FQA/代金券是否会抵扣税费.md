# 代金券是否会抵扣税费？


代金券是否会抵扣税费分两种情况：

一、商家在APOS后台 运营设置--门店设置--业务设置--跨境销售，保税品定价含跨境综合税设置为“是”

则在商城里商品显示的价格是商品原价+税费部分价格，这时下单使用代金券，抵扣订单金额抵扣的商品已含税金额，因此是抵扣了原商品金额和对应的税费部分金额。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E4%BB%A3%E9%87%91%E5%88%B8%E6%98%AF%E5%90%A6%E4%BC%9A%E6%8A%B5%E6%89%A3%E7%A8%8E%E8%B4%B9.md%2F97e9dc76-3a23-46dc-bcaf-f0d6963d8f19_image.png?Expires=1824881390&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=w6xn%2Furl9Lgw58CLp0fGojAlGoA%3D&x-oss-process=image%2Fformat%2Cpng)

二、如果在上图设置中设置保税品定价含跨境综合税设置为“否”。

那么商城订单页可见，下单时，会在商品标价之外额外再计算一笔税费。

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E4%BB%A3%E9%87%91%E5%88%B8%E6%98%AF%E5%90%A6%E4%BC%9A%E6%8A%B5%E6%89%A3%E7%A8%8E%E8%B4%B9.md%2F8a9a864c-67e8-42ea-9cb6-8a28f52bb19a_image.png?Expires=1824881799&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=h1%2FiPeHtJcBS%2B7lgh2yx6duv5OM%3D&x-oss-process=image%2Fformat%2Cpng)

这时如果使用代金券，代金券将抵扣商品金额+运费部分，不会抵扣税费。

如下图，和上图对比，选了100元代金券后，比上图订单金额减少100，税费不变。
