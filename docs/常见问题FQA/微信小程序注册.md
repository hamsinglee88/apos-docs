# 微信小程序注册方法


分为两种情况，

第一种：直接注册。需要微信认证，审核要3天左右

第二种：已有认证过的微信服务号，在微信服务号快速注册同主体的小程序（可免认证）半小时内可以弄好

为方便公众号快捷接入小程序，并在各功能中使用小程序的服务，推荐商家直接复用公众号资质注册小程序，优点有三：

1、无须重新提交主体材料;

2、 无须对公账户打款;

3、无须支付300元认证费用。

第一种小程序注册方法

> **第一步：**百度**“微信公众号”**，进入到微信公众号平台注册页面，点击**“立即注册”。**

第二种小程序注册方法

> 如果你的公众号是服务号且已经认证，可以使用**“快速注册认证小程序流程”**，无需重新提交主体材料、无需对公打款、无需支付300元认证费用。
>
> **第一步：**登录微信公众号平台，进入已有的公众号账号，点击**“小程序管理”**

**第二步：**进入公众号管理页面，点击“添加”选择“快速注册并认证小程序”。
