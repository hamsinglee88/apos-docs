# 按订单金额区间计算运费


## 操作步骤：

## 一、找到菜单路径

登录管理后台后，到右上角应用商店--商城--业务设置--移动商城设置--物流配送方式设置

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E6%8C%89%E8%AE%A2%E5%8D%95%E9%87%91%E9%A2%9D%E5%8C%BA%E9%97%B4%E7%AE%97%E8%BF%90%E8%B4%B9.md%2F3d1aae80-3202-4652-b9bc-6c640a6ceaa8_image.png?Expires=1824351187&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=KpZRZN3s1VnS88xt36NKV1OoHT8%3D&x-oss-process=image%2Fformat%2Cpng)

## 二、编辑订单区间运费

点击设置，比如设置订单商品金额在0-89元时，要10元费用，大于89元时，0运费。

设置完可以下不同价格商品的本地单测试验证。
