# 运费模板


## 基本介绍：

运费模板用于提供商城下单时计算商品的运费，目前仅用于本地配送类型订单。

模板可设置按首件续件、按首重续重模式计算运费。下单多商品使用不同的模板时，可设置按模板维度计算运费，或者是下单所有商品中一个作为首件/首重其他作为续件/续重计算运费；每个模板下可以设置多个配送区域，不同区域下设置不同的运费以区分不同地区运费

## 使用介绍：

### 1.默认模板

每个商户在创建商户时会自动创建一个默认模板。模板的配送区域包含当前热门国家，

按件计费。当商品没有关联一个具体的模板时，自动应用商户下的默认模板。

### 2.新建运费模板

在商户管理后台--商品管理--配送设置页可新建运费模板。

### 5.运费模板计算运费规则：

1. 新建按件计费的模板：设置首件运费，续件运费，当下单商品不足首件数量时，运费为首件运费，当前下单商品超过首件不足续件数量时，运费为首件+续件运费

   举例： 商品A应用模板一，模板一配置配送区域到中国，首件2 运费5
   ，续件3 运费1，下单1件A，运费为5，下单3件A，运费=5+1=6
2. 新建按重计费的模板：设置首重运费，续重运费，当下单商品不足首重时，运费为首重运费，当前下单商品超过首件不足续重时，运费为首重+续重运费

   如果使用按重计费，务必维护好商品重量基础信息

   举例： 商品A应用模板一，模板一配置配送区域到中国，首重2000 g 运费5
   ，续重3000 g 运费1，下单1000 g的A，运费为5，下单3000
   g的A，运费=5+1=6
3. 模板下配置多个计费规则，一个模板下如果已经配置了某个区域的计费规则，新增一条规则时，不能选择重复的区域。用户下单时，会根据收件人地区匹配下单商品应用的模板配置的地区计算相应运费

   举例：商品A应用模板一，模板一下配置配送区域有广东省，首件2运费为1，续件1 运费1， 有云南省 首件1 运费为2，续件1运费为2，则用户下单购买一件A，收件人地区在广东时运费为1，收件人地区在云南时运费为2

   4. 模板运费计算方式设置为“组合运费“
      ![](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fcodyxiong%40jieztech.com%2F%E8%BF%90%E8%B4%B9%E6%A8%A1%E6%9D%BF.docx%2F691bbd5d-7b09-48b1-bd3f-2c9c5f0dd593_image4.png?Expires=1786498950&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=GiF0Yg766d4naGQArFggQsiwduY%3D&x-oss-process=image%2Fformat%2Cpng)

   此时一次下单多商品时，如果不同商品用了不同的模板，取各模板中首重/首件费用最大者为首重/首件，其他按照续重/续件计算运费

   举例：
   商品A应用模板一，单个A商品重量为1000g,模板一时按重计费，首重1000g
   运费为6 续重1000g运费为5， 商品B应用模板二，模板二按件计费，首件2
   运费为5，续件1 运费为2，此时在商城同时下单3件A和3件B，其中一件A作为首重，2件A作为续重计算运费，A的运费为6+5+5=16，B全部算作续件，B的运费为2+2+2=6，这个订单的运费为16+6=22
4. 模板运费计算方式设置为“按商品累加运费”
   ![](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fcodyxiong%40jieztech.com%2F%E8%BF%90%E8%B4%B9%E6%A8%A1%E6%9D%BF.docx%2F232b46c4-5819-4933-bfdc-859978d1daf4_image5.png?Expires=1786498950&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=ReMKMfBdXClmAB8Or6e7Kulq5e8%3D&x-oss-process=image%2Fformat%2Cpng)

   此时一次下单多商品时，如果不同商品使用了不同模板，以模板维度，每个模板下计算一个首件/首重，其他作为续件/续重计算运费

   举例：商品A应用模板一，单个A商品重量为1000g,模板一时按重计费，首重1000g
   运费为6 续重1000g运费为5， 商品B应用模板二，模板二按件计费，首件2
   运费为5，续件1 运费为2，商品C也应用模板二。此时同时下单3件和3件B和3件C

   订单中模板一下A的运费为一个首重+2个续重，运费为6+5+5=16；模板二下有3件B三件C，共计一个首件，其他作为续件，模板二运费为5+1+1+1+1=9。总的运费为25
