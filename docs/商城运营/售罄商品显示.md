# 售罄商品显示


基本介绍：商户后台中当设置不允许零库存出库情况下 ，该商品的库存为零时，商品主图页面会显示暂无库存。

进入商户后台--系统管理--系统设置--开启售罄商品显示

商城中展示效果

当商品没有库存情况，商城中商品会显示暂无库存，商品库存正常的情况，商城中商品正常显示。
