# 设置收货后可申请售后时间


基本介绍：订单确认收货后在N天内可发起售后，超过N天售后入口将隐藏。为0或为空表示所有时间均可发起售后。

进入商户后台--系统管理--系统设置--收货后可申请售后时间。
