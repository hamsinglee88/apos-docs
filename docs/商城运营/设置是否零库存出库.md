# 设置是否允许零库存出库

基本介绍：
设置不允许零库存出库情况下，当该商品库存为零时，用户在商城下单时会合理提示。
设置允许零库存出库情况下，当该商品库存为零时，用户在商城可以正常下单，商城库存为负数。

进入商户后台--系统管理--系统设置--设置零库存出库开关

门店管理--门店设置--业务设置---设置零库存出库
以门店设置-业务设置中的零库存出库的设置开关为准

![image.png](http://media-hk01.oss-cn-hongkong.aliyuncs.com/images%2Fjackli%40jieztech.com%2F%E8%AE%BE%E7%BD%AE%E6%98%AF%E5%90%A6%E5%85%81%E8%AE%B8%E9%9B%B6%E5%BA%93%E5%AD%98%E5%87%BA%E5%BA%93.md%2F7680aff7-c87b-41f2-bf7d-410fc2b3ec32_image.png?Expires=1815448374&OSSAccessKeyId=LTAIXyAB35iFpHM0&Signature=vYJ8uVvDo6Gco2bw%2BVH5Rpjd7wo%3D&x-oss-process=image%2Fformat%2Cpng)

商城中展示的效果：

设置不允许零库存出库情况下，用户在商城中点击该没有库存的商品，会合理提示 商品库存不足，
设置允许零库存出库情况下，用户在商城中点击没有库存的商品，可以正常下单支付，
