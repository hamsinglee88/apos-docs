# 发货后自动确认收货

> 文档地址：商城运营/发货后自动确认收货.md


基本介绍：用户在商城支付订单，已商家发货时间为准，发货N天时间后自动确认收货。

进入商户后台--系统管理--系统设置--设置发货后自动确认收货时间
